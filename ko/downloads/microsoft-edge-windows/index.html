Microsoft Edge는 2015

다운로드
안전하고 무료이며 빠름

Microsoft Edge는 2015년에 처음 출시된 Microsoft에서 개발한 웹 브라우저입니다. 이전 버전의 Windows에서 기본 웹 브라우저였던 Internet Explorer에 대한 현대적이고 빠른 대안으로 설계되었습니다.

Microsoft Edge의 두드러진 특징은 속도입니다. 최신 웹 표준에 최적화된 EdgeHTML이라는 새로운 렌더링 엔진을 사용하여 다른 브라우저보다 더 빠르고 효율적으로 페이지를 로드할 수 있습니다. Edge에는 산만함을 제거하고 읽고 있는 콘텐츠에 더 쉽게 집중할 수 있게 해주는 읽기 모드와 같은 기능이 있는 깨끗하고 단순한 인터페이스가 있습니다.

성능 및 디자인과 함께 Microsoft Edge는 웹을 탐색하는 동안 사용자를 보호하기 위한 다양한 보안 및 개인 정보 보호 기능도 제공합니다. 여기에는 팝업 및 원치 않는 쿠키를 차단하는 옵션과 강력한 암호를 생성하고 나중에 사용할 수 있도록 저장할 수 있는 내장형 암호 관리자가 포함됩니다.

Microsoft Edge의 또 다른 주목할만한 기능은 Microsoft의 다른 제품 및 서비스와의 통합입니다. 예를 들어 브라우저를 사용하여 OneDrive 계정에 액세스하거나 주석 도구를 사용하여 웹 페이지를 표시하고 다른 사람과 공유할 수 있습니다. Microsoft Edge는 브라우저에 추가 기능을 추가하는 작은 프로그램인 “확장 프로그램"도 지원합니다.

요약하자면 Microsoft Edge는 빠르고 안전하며 기능이 풍부한 웹 브라우저로 개인용 및 전문가용 모두에 적합합니다. 여가를 위해 웹을 탐색하든 중요한 문서 작업을 하든 Edge는 작업을 완료하는 데 필요한 도구와 기능을 갖추고 있습니다.

기능 및 하이라이트

사용자 인터페이스: Microsoft Edge는 깨끗하고 현대적이며 탐색하기 쉬운 사용자 친화적인 인터페이스와 함께 제공됩니다. 미니멀한 디자인으로 사용자가 보고 있는 콘텐츠에 쉽게 집중할 수 있습니다.

탭 관리: Edge는 뛰어난 탭 관리 기능을 제공합니다. 키보드 단축키, 마우스 클릭 또는 터치 제스처를 사용하여 탭을 쉽게 열고 닫고 전환할 수 있습니다. 또한 현재 사용하지 않지만 나중에 액세스하려는 탭을 별도로 설정할 수 있습니다.

읽기용 보기: Microsoft Edge는 기사의 본문과 이미지만 남기고 웹 페이지에서 광고와 방해 요소를 제거하는 “읽기용 보기” 옵션을 제공합니다. 이 기능을 사용하면 웹 페이지를 더 쉽고 덜 산만하게 읽을 수 있습니다.

Cortana 통합: Cortana는 Microsoft의 가상 도우미이며 Edge는 Cortana와 통합되어 사용자에게 추가 기능을 제공합니다. 예를 들어 웹 사이트를 찾거나 주제에 대한 정보를 얻을 수 있도록 Cortana에 요청할 수 있습니다.

확장: Edge는 확장을 지원하여 사용자가 브라우저에 추가 기능을 추가할 수 있도록 합니다. 광고 차단, 암호 관리, 소셜 미디어 등을 위한 확장 프로그램이 있습니다.

PDF 지원: Edge는 추가 소프트웨어 없이도 PDF 파일을 보고 주석을 달 수 있습니다. 이 기능은 PDF 문서로 자주 작업해야 하는 사용자에게 특히 유용합니다.

보안: Microsoft Edge에는 악의적인 웹 사이트 및 공격으로부터 사용자를 보호하는 데 도움이 되는 몇 가지 보안 기능이 있습니다. 내장형 피싱 필터, 맬웨어 보호 및 운영 체제에서 브라우저를 분리하는 샌드박싱 기술이 있습니다.

사용자 인터페이스

Microsoft Edge 사용자 인터페이스는 단순성과 기능에 중점을 두고 깨끗하고 현대적이며 사용자 친화적으로 설계되었습니다. UI의 핵심 요소는 다음과 같습니다.

주소 표시줄: 브라우저 창 상단에 있는 주소 표시줄을 통해 사용자는 URL 또는 검색어를 입력할 수 있습니다. Edge는 Bing을 기본 검색 엔진으로 사용하지만 사용자는 설정에서 선호하는 검색 엔진으로 변경할 수 있습니다.

탭: 주소 표시줄 위에 있는 탭 인터페이스를 사용하여 열려 있는 여러 웹 페이지를 관리할 수 있습니다. 각 탭에는 페이지 제목이 표시되며 사용 가능한 경우 쉽게 식별할 수 있는 파비콘이 표시됩니다.

새 탭 페이지: 새 탭을 열면 검색 표시줄, 자주 방문하는 웹 사이트, 뉴스 및 날씨 업데이트와 같은 개인화된 콘텐츠를 표시하는 사용자 지정 가능한 시작 페이지가 사용자를 맞이합니다.

탐색 버튼: 주소 표시줄 왼쪽에 있으며 사용자는 쉽게 탐색할 수 있는 표준 뒤로, 앞으로 및 새로 고침 버튼을 찾을 수 있습니다.

메뉴: 창의 오른쪽 상단 모서리에 있는 점 3개 아이콘(줄임표)을 클릭하면 북마크, 기록, 확장 프로그램, 개인 정보 및 보안 옵션 등과 같은 다양한 브라우저 기능 및 설정에 액세스할 수 있는 기본 메뉴가 열립니다. .

컬렉션: Microsoft Edge의 고유한 기능인 컬렉션을 사용하면 링크, 이미지, 텍스트와 같은 웹 콘텐츠를 전용 사이드바에 수집하고 구성할 수 있습니다. 이 기능은 주소 표시줄 오른쪽에 있는 컬렉션 아이콘을 클릭하여 액세스할 수 있습니다.

즐겨찾기 및 읽기 목록: 사용자는 나중에 쉽게 액세스할 수 있도록 즐겨찾는 웹 사이트를 저장할 수 있으며 읽기 목록 기능을 사용하면 기사나 웹 페이지를 저장하여 나중에 또는 오프라인에서 읽을 수 있습니다.

확장 프로그램: Microsoft Edge는 브라우저 기능을 향상하고 사용자 환경을 사용자 지정할 수 있는 Chrome 웹 스토어의 확장 프로그램을 포함하여 다양한 확장 프로그램을 지원합니다.

접근성 기능: Edge는 웹 페이지 레이아웃을 단순화하고 텍스트 음성 변환 기능을 제공하는 몰입형 리더 모드, 시각 장애가 있는 사용자를 지원하는 고대비 테마와 같은 다양한 접근성 기능을 제공합니다.

Microsoft Edge의 사용자 인터페이스는 모든 경험 수준의 사용자가 쉽게 탐색할 수 있는 원활하고 효율적인 검색 환경을 제공하도록 설계되었습니다.

탭 관리

Microsoft에서 개발한 웹 브라우저인 Microsoft Edge는 탭 관리를 포함하여 사용자 경험을 개선하기 위한 몇 가지 기능을 도입했습니다. Microsoft Edge의 탭 관리는 사용자가 열려 있는 여러 탭을 효과적으로 구성, 탐색 및 처리할 수 있도록 설계되었습니다.

Microsoft Edge 탭 관리 기능의 몇 가지 주요 측면은 다음과 같습니다.

세로 탭: 이 기능을 사용하면 사용자가 열려 있는 탭을 기존의 가로 레이아웃에서 창 측면의 세로 스택으로 이동할 수 있습니다. 이렇게 하면 열려 있는 탭에 대한 더 나은 개요를 제공하여 탭을 더 쉽게 찾고 전환할 수 있습니다. 세로 탭을 활성화하려면 사용자는 브라우저 창의 왼쪽 상단 모서리에 있는 “세로 탭” 버튼을 클릭하기만 하면 됩니다.

탭 그룹: 탭 그룹을 사용하면 사용자가 단일 그룹에서 유사한 탭을 분류하고 레이블을 지정할 수 있습니다. 이렇게 하면 브라우저를 체계적으로 유지하고 관련 탭 사이를 더 쉽게 탐색할 수 있습니다. 탭 그룹을 만들려면 사용자가 탭을 마우스 오른쪽 버튼으로 클릭하고 “새 그룹에 탭 추가” 또는 “기존 그룹에 탭 추가"를 선택할 수 있습니다.

탭 미리보기: 이 기능은 탭 위로 마우스를 가져가면 탭 내용의 작은 미리보기를 제공합니다. 이를 통해 사용자는 탭으로 전환할 필요 없이 탭의 콘텐츠를 빠르게 식별할 수 있습니다.

수면 탭: Microsoft Edge는 시스템 리소스를 절약하는 데 도움이 되는 “수면 탭"이라는 기능을 도입했습니다. 일정 시간 동안 탭을 사용하지 않으면 자동으로 절전 모드로 전환되어 시스템 리소스를 확보합니다. 사용자는 탭이 절전 모드로 전환되기 전의 시간을 사용자 정의할 수 있으며 절전 탭은 클릭 시 자동으로 다시 로드됩니다.

탭 고정: 사용자는 자주 방문하는 웹사이트나 중요한 탭을 고정하여 탭 행의 시작 부분에 영구적으로 표시되도록 할 수 있습니다. 이렇게 하면 가장 중요하거나 자주 사용하는 탭에 쉽게 액세스할 수 있습니다.

바로 가기 키: Microsoft Edge는 새 탭을 여는 Ctrl+T, 현재 탭을 닫는 Ctrl+W, 열려 있는 탭 간을 전환하는 Ctrl+Tab 등 탭 관리를 위한 다양한 바로 가기 키를 지원합니다.

Microsoft Edge의 이러한 탭 관리 기능은 전체 검색 환경을 크게 개선하여 한 번에 여러 탭을 관리하는 사용자를 위해 보다 효율적이고 체계적으로 만듭니다.

읽기용 보기

Microsoft Edge의 읽기용 보기 기능은 사용자의 온라인 읽기 환경을 개선하도록 설계된 편리한 도구입니다. 광고, 팝업 및 기타 중요하지 않은 요소와 같은 주의를 산만하게 하는 요소를 제거하여 웹 페이지의 레이아웃을 단순화하여 웹 페이지의 주요 콘텐츠에 더 쉽게 집중할 수 있도록 합니다.

다음은 Microsoft Edge의 읽기용 보기 기능의 몇 가지 주요 측면입니다.

깔끔한 레이아웃: 읽기용 보기는 광고, 사이드바 및 기타 복잡한 요소를 제거하여 사용자가 읽고 있는 텍스트에 집중할 수 있도록 깔끔하고 깔끔한 인터페이스를 제공합니다. 이는 긴 기사나 조밀한 콘텐츠를 읽을 때 특히 유용할 수 있습니다.

사용자 지정: 사용자는 다양한 글꼴 스타일, 텍스트 크기 및 배경색 중에서 선택하여 읽기용 보기의 모양을 사용자 지정할 수 있습니다. 이를 통해 개인의 선호도와 접근성 요구 사항을 충족하는 개인화된 독서 경험이 가능합니다.

텍스트 음성 변환: 이 기능은 콘텐츠를 소리내어 읽어주는 “소리내어 읽기"라는 내장 텍스트 음성 변환 기능을 제공합니다. 이는 시각 장애가 있는 사용자나 콘텐츠를 읽는 것보다 듣는 것을 선호하는 사용자에게 도움이 될 수 있습니다.

오프라인 읽기: 읽기용 보기를 사용하면 사용자가 오프라인 읽기를 위해 웹 페이지를 저장할 수 있으므로 인터넷 연결을 사용할 수 없는 경우에도 콘텐츠에 편리하게 액세스할 수 있습니다.

쉬운 액세스: 읽기용 보기 기능은 주소 표시줄에 있는 책 아이콘을 사용하거나 키보드 단축키 Ctrl+Shift+R을 눌러 쉽게 켜고 끌 수 있습니다. 이렇게 하면 필요에 따라 표준 보기와 단순화된 보기 사이를 간단하게 전환할 수 있습니다.

Microsoft Edge의 읽기용 보기는 온라인 콘텐츠를 읽을 때 사용자 경험을 향상시키는 유용한 도구입니다. 사용자 지정 옵션과 접근성 기능을 제공함으로써 광범위한 사용자 요구와 선호도를 충족합니다.

코타나 통합

Microsoft Edge는 회사의 가상 비서인 Cortana를 통합하여 사용자 경험을 향상하고 보다 원활한 브라우징 경험을 제공합니다. Cortana 통합 기능으로 알려진 이 통합을 통해 사용자는 Edge 브라우저 내에서 가상 도우미와 직접 상호 작용할 수 있습니다. Cortana는 처음에 Windows 10과 함께 도입되었지만 이후 다른 플랫폼 및 장치에서도 사용할 수 있게 되었습니다.

Microsoft Edge의 Cortana 통합은 다음과 같은 여러 이점과 기능을 제공합니다.

음성 검색: 사용자는 주소 표시줄에서 직접 Cortana를 사용하여 음성 검색을 수행할 수 있습니다. 이 기능을 사용하면 사용자가 검색어를 수동으로 입력하지 않고도 정보를 검색할 수 있습니다.

상황 정보: Cortana는 웹 페이지에서 선택한 텍스트에 대한 상황 정보를 제공할 수 있습니다. 텍스트를 강조 표시하고 마우스 오른쪽 버튼을 클릭하면 사용자는 현재 페이지를 떠나지 않고도 Cortana의 지원에 액세스하고 관련 정보 또는 정의를 얻을 수 있습니다.

개인화된 제안: Cortana는 사용자의 검색 기록, 기본 설정 및 관심사를 기반으로 개인화된 제안을 제공할 수 있습니다. 이를 통해 사용자는 새로운 콘텐츠를 발견하고 특정 정보를 검색할 때 시간을 절약할 수 있습니다.

미리 알림 및 알림: Cortana는 Microsoft Edge 브라우저를 통해 미리 알림 및 알림을 직접 보낼 수 있습니다. 사용자는 별도의 앱이나 창을 열 필요 없이 미리 알림을 설정하거나 중요한 알림을 받을 수 있습니다.

다른 Microsoft 서비스와의 통합: Cortana는 Outlook, OneNote 및 To-Do와 같은 다른 Microsoft 서비스와 원활하게 작동하므로 사용자가 브라우저에서 직접 작업, 약속 및 메모를 관리할 수 있습니다.

장치 간 동기화: Cortana는 장치 간에 정보를 동기화할 수 있으므로 사용자는 스마트폰, 태블릿 또는 PC에서 미리 알림, 메모 및 기타 데이터에 액세스할 수 있습니다.

이후 Microsoft는 독립 실행형 가상 비서로서의 Cortana에서 Microsoft 에코시스템 내의 보다 구체적인 생산성 시나리오로 통합하는 것으로 초점을 이동했다는 점에 유의해야 합니다. 결과적으로 가상 비서의 가용성 및 기능은 지역 및 장치에 따라 다를 수 있습니다.

그러나 Microsoft Edge의 Cortana 통합 기능은 여전히 ​​사용자에게 다양한 이점과 편의성을 제공하여 전반적인 브라우징 경험을 향상시킵니다.

확장 지원

확장 프로그램은 브라우저의 기능을 수정하고 향상시킬 수 있는 작은 소프트웨어 프로그램으로, 사용자가 자신의 필요와 선호도에 따라 브라우징 경험을 조정할 수 있습니다.

확장에 대한 Microsoft Edge의 지원은 다음과 같은 몇 가지 이점을 제공합니다.

광범위한 확장: Microsoft Edge는 Chromium을 기반으로 하므로 Chrome 웹 스토어에서 사용할 수 있는 대부분의 Chrome 확장을 지원할 수 있습니다. 이를 통해 사용자는 자신의 필요에 맞는 확장 프로그램을 쉽게 찾고 설치할 수 있습니다. 선택할 수 있는 크고 확립된 옵션 라이브러리가 이미 있기 때문입니다.

간편한 설치: Microsoft Edge에 확장 프로그램을 설치하는 것은 매우 간단합니다. 사용자는 Microsoft Edge 추가 기능 웹사이트 또는 Chrome 웹 스토어를 방문하여 확장 프로그램을 찾아보고 설치할 수 있습니다. 원하는 확장 프로그램을 찾으면 사용자는 “Chrome에 추가” 또는 “받기” 버튼을 클릭하여 설치할 수 있으며 확장 프로그램이 브라우저에 추가됩니다.

확장 관리: Microsoft Edge는 설치된 확장을 관리하기 위한 사용하기 쉬운 인터페이스를 제공합니다. 사용자는 브라우저 오른쪽 상단 모서리에 있는 점 3개 메뉴 아이콘을 클릭하고 “확장 프로그램"을 선택하여 확장 프로그램 페이지에 액세스할 수 있습니다. 여기에서 사용자는 확장을 활성화, 비활성화 또는 제거할 수 있을 뿐만 아니라 개별 확장에 대한 설정에 액세스할 수 있습니다.

보안 및 개인 정보: Microsoft는 사용자의 보안 및 개인 정보를 보장하기 위해 최선을 다하고 있으며 이는 확장에 대한 지원까지 확장됩니다. Microsoft Edge에는 추가 기능 웹 사이트에 제출된 확장에 대한 검토 프로세스가 있어 사용자에게 제공되기 전에 특정 표준을 충족하는지 확인합니다. 또한 사용자는 각 확장 프로그램에 부여된 권한을 제어하여 액세스할 수 있는 데이터를 제한할 수 있습니다.

개발자 지원: Microsoft는 Microsoft Edge용 확장을 만드는 데 관심이 있는 개발자를 위한 리소스 및 설명서를 제공합니다. 개발자는 Microsoft Edge 확장 도구 키트를 사용하여 기존 Chrome 확장 프로그램을 변환하거나 처음부터 새 확장 프로그램을 만들 수 있습니다. 이 지원은 확장 생태계의 성장을 장려하고 사용자가 다양한 도구 및 기능에 액세스할 수 있도록 합니다.

PDF 지원

Microsoft Edge의 PDF 기능은 타사 소프트웨어 없이도 PDF 파일을 보고, 탐색하고, 주석을 달고, 관리할 수 있는 편리한 방법을 사용자에게 제공합니다. 다음은 PDF 문서 작업을 위해 Microsoft Edge에서 제공하는 몇 가지 주요 기능입니다.

PDF 보기: Microsoft Edge를 사용하면 사용자가 브라우저 내에서 직접 PDF 파일을 열고 볼 수 있습니다. 파일은 로컬 드라이브 또는 URL을 통해 액세스할 수 있습니다.

탐색: 브라우저는 문서 축소판 보기를 위한 사이드바, 목차, 책갈피, 페이지 번호를 입력하여 특정 페이지로 이동하는 기능과 같은 직관적인 탐색 도구를 제공합니다.

확대/축소 및 보기 옵션: Edge는 가독성을 높이고 사용자 기본 설정을 수용하기 위해 단일 페이지, 연속 스크롤 및 두 페이지 보기와 같은 다양한 확대/축소 수준 및 보기 옵션을 제공합니다.

주석: 사용자는 주석, 강조 표시, 밑줄 및 취소선을 PDF 텍스트에 추가하여 문서를 읽는 동안 쉽게 공동 작업을 하거나 메모를 작성할 수 있습니다. 또한 사용자는 PDF에 모양과 텍스트 상자를 그리거나 추가할 수 있습니다.

텍스트 선택 및 복사: Microsoft Edge를 사용하면 PDF 파일에서 텍스트를 선택하고 복사한 다음 다른 응용 프로그램에 붙여넣을 수 있습니다.

검색: 브라우저에는 사용자가 PDF 문서 내에서 특정 텍스트나 구를 찾을 수 있는 내장 검색 기능이 포함되어 있습니다.

양식 작성 및 서명: Edge는 PDF 양식 작성 및 서명을 지원하므로 사용자가 브라우저 내에서 직접 다양한 유형의 양식을 작성하고 제출할 수 있습니다.

접근성: Microsoft Edge는 화면 판독기와 같은 보조 기술에 대한 기본 제공 지원을 제공하며 시각 장애가 있는 사용자의 접근성을 개선하기 위해 고대비 모드를 활성화하거나 소리내어 읽을 수 있는 옵션을 제공합니다.

저장 및 인쇄: 사용자는 PDF 파일을 로컬 드라이브에 쉽게 저장하거나 브라우저에서 직접 인쇄할 수 있습니다.

다른 Microsoft 서비스와의 통합: Edge는 OneDrive, SharePoint 및 Outlook과 같은 다른 Microsoft 서비스와 원활하게 통합되어 다양한 플랫폼에서 PDF 파일을 쉽게 공유, 액세스 및 관리할 수 있습니다.

Microsoft Edge는 PDF 문서 작업을 위한 강력한 기능 집합을 제공하지만 전용 PDF 소프트웨어 또는 타사 브라우저 확장이 제공하는 모든 고급 기능을 포함하지 않을 수 있습니다. 그러나 대부분의 사용자에게 Edge의 내장 PDF 지원은 일상적인 작업과 문서 관리에 충분합니다.

보안 기능

Microsoft Edge는 사용자에게 안전한 브라우징 경험을 보장하기 위해 보안에 중점을 둡니다. Microsoft Edge에 포함된 주요 보안 기능 중 일부는 다음과 같습니다.

SmartScreen 필터: SmartScreen은 피싱 공격, 맬웨어 및 기타 악성 소프트웨어로부터 사용자를 보호하는 데 도움이 되는 Microsoft에서 개발한 기능입니다. 지속적으로 업데이트되는 보고된 악성 사이트 및 다운로드 목록에 대해 웹 사이트 및 파일의 URL을 확인하여 잠재적으로 위험한 웹 사이트를 방문하거나 잠재적으로 유해한 파일을 다운로드하려는 경우 사용자에게 경고합니다.

샌드박싱: Edge는 웹 콘텐츠를 샌드박스 처리된 프로세스로 분리하는 다중 프로세스 아키텍처를 사용합니다. 즉, 악의적인 웹 사이트가 브라우저의 취약성을 악용하는 경우 손상이 제한되고 공격자가 사용자의 개인 데이터 또는 기타 시스템 리소스에 액세스할 수 없습니다.

Microsoft Defender 통합: Microsoft Edge는 회사의 바이러스 백신 및 맬웨어 방지 솔루션인 Microsoft Defender와 기본적으로 통합되어 있습니다. 이 통합은 잠재적으로 유해한 파일을 감지하고 차단하는 데 도움이 되도록 브라우저를 통해 액세스하는 파일 및 다운로드에 대한 실시간 검사를 제공합니다.

추적 방지: Microsoft Edge는 사용자가 온라인 개인 정보를 더 잘 제어할 수 있도록 추적 방지 옵션을 제공합니다. 사용자는 기본, 균형 또는 엄격의 세 가지 추적 방지 수준 중에서 선택할 수 있습니다. 각 수준은 최고 수준의 개인 정보 보호를 제공하는 엄격한 설정으로 다양한 범주의 추적기를 차단합니다.

InPrivate 브라우징: 다른 브라우저의 프라이빗 브라우징 모드와 마찬가지로 Edge의 InPrivate 브라우징을 사용하면 브라우징 기록, 임시 인터넷 파일 또는 쿠키가 장치에 저장되지 않고 웹을 브라우징할 수 있습니다.

보안 업데이트: Microsoft Edge는 보안 패치 및 개선 사항으로 정기적으로 업데이트됩니다. 이러한 업데이트는 Windows 업데이트를 통해 안전하게 제공되므로 최신 위협으로부터 사용자를 보호할 수 있습니다.

암호 모니터: 이 기능은 저장된 암호가 알려진 데이터 유출과 관련된 경우 사용자에게 경고하여 온라인 계정을 안전하게 유지하는 데 도움이 됩니다. 그런 다음 계정 보안을 유지하기 위해 영향을 받는 암호를 변경하라는 메시지가 사용자에게 표시됩니다.

기본적으로 HTTPS: Microsoft Edge는 사용 가능한 경우 웹사이트 연결을 HTTP에서 HTTPS로 자동 업그레이드하여 사용자 데이터를 보호하기 위해 보다 안전하고 암호화된 연결을 제공합니다.

자주 묻는 질문

마이크로소프트 엣지란?

Microsoft Edge는 Windows, macOS, Android 및 iOS에서 사용할 수 있는 Microsoft에서 개발한 웹 브라우저입니다.

Microsoft Edge는 언제 처음 출시되었습니까?

Microsoft Edge는 Windows 10의 일부로 2015년 7월 29일에 처음 출시되었습니다.

Microsoft Edge의 전신은 무엇입니까?

Microsoft Edge의 전신은 Internet Explorer였습니다.

Microsoft Edge에는 어떤 기능이 있습니까?

Microsoft Edge의 일부 기능에는 사용자 지정 가능한 시작 페이지, 확장, 읽기 모드, 내장 PDF 리더 및 Cortana(Microsoft의 가상 비서)와의 통합이 포함됩니다.

모바일 장치에서 Microsoft Edge를 사용할 수 있습니까?

예, Microsoft Edge는 Android 및 iOS를 포함한 모바일 장치에서 사용할 수 있습니다.

Microsoft Edge의 기본 검색 엔진은 무엇입니까?

Microsoft Edge의 기본 검색 엔진은 Bing이지만 원하는 경우 사용자가 다른 검색 엔진으로 변경할 수 있습니다.

Microsoft Edge의 읽기 목록 기능은 무엇입니까?

Microsoft Edge의 읽기 목록 기능을 사용하면 나중에 읽을 수 있도록 웹 페이지를 저장할 수 있습니다.

Microsoft Edge의 “허브"란 무엇입니까?

Microsoft Edge의 허브는 책갈피, 읽기 목록, 기록 및 다운로드에 대한 액세스를 제공하는 사이드바입니다.

Microsoft Edge는 확장을 지원합니까?

예, Microsoft Edge는 Microsoft Store 에서 다운로드할 수 있는 확장을 지원합니다.

Microsoft Edge는 장치 간에 데이터를 동기화할 수 있나요?

예, Microsoft Edge는 책갈피, 기록, 암호 및 양식 데이터를 포함하여 장치 간에 데이터를 동기화할 수 있습니다.

Microsoft Edge의 컬렉션 기능은 무엇입니까?

Microsoft Edge의 컬렉션 기능을 사용하면 사용자가 웹 콘텐츠를 그룹으로 저장하고 구성할 수 있습니다.

Microsoft Edge는 웹 표준과 호환되나요?

예, Microsoft Edge는 HTML5, CSS3 및 JavaScript를 포함한 웹 표준과 호환됩니다.

Microsoft Edge DevTools란 무엇입니까?

Microsoft Edge DevTools는 개발자가 웹 페이지를 디버깅하고 테스트하는 데 사용할 수 있는 도구 세트입니다.

Microsoft Edge가 광고를 차단할 수 있나요?

예, Microsoft Edge는 내장된 추적 방지 기능을 사용하여 광고를 차단할 수 있습니다.

Microsoft Edge는 무료로 사용할 수 있나요?

예, Microsoft Edge는 무료로 사용할 수 있으며 Windows 장치에 사전 설치되어 제공됩니다.

추가 정보

Microsoft Edge는 Google Chrome의 기반이기도 한 오픈 소스 브라우저 프로젝트인 Chromium을 기반으로 합니다. 2018년 12월 Microsoft는 호환성과 성능을 개선하기 위해 Chromium 엔진을 사용하여 Edge를 재구축할 것이라고 발표했습니다. 새로운 버전의 Edge는 2020년 1월에 출시되었으며 이제 Google Chrome과 동일한 핵심 기술을 기반으로 합니다.

Microsoft Edge와 Google Chrome은 Chromium에서 공통 기반을 공유하지만 여전히 고유한 기능과 디자인을 가진 별개의 브라우저입니다. 예를 들어 Edge에는 Chrome에는 없는 읽기 모드와 주석 도구가 포함되어 있으며 Microsoft의 다른 제품 및 서비스와도 통합됩니다. 그러나 두 브라우저는 동일한 확장 프로그램과 웹 애플리케이션을 실행할 수 있으며 성능 및 보안 기능도 비슷합니다.

브라우저 전투: 최고의 인터넷 네비게이터에 대한 재치 있는 리뷰

광대하고 무한한 인터넷 영역을 탐색하기 위해 의존하는 신뢰할 수 있는 도구입니다. 고양이 비디오를 검색하든 최신 피젯 스피너를 구매하든 웹 브라우저는 필수적입니다. 그러나 솔직히 말해서 모두가 평등하게 창조된 것은 아닙니다. 일부는 매끄럽고 빠르며 다른 일부는 투박하고 일요일 아침의 달팽이처럼 느립니다.

먼저 전능한 Google 크롬이 있습니다. 그것은 언덕의 왕, 최고의 개, 브라우저의 비욘세입니다. Chrome은 매우 빠르며 인터넷 문제를 해결하는 데 필요한 모든 기능을 갖추고 있습니다. 또한, 인터넷이 끊길 때 나타나는 공룡 게임이 있는 유일한 브라우저입니다. 브라우저가 그렇게 할 수 있습니까? 나는 그렇게 생각하지 않았다.

다음으로 Mozilla Firefox가 있습니다. 항상 Chrome을 따라잡으려는 허접한 약자와 같습니다. 물론 빠르지는 않지만 마음이 있습니다. Firefox는 오픈 소스입니다. 즉, 모든 사람이 무료로 액세스할 수 있는 브라우저를 만드는 데 열정을 가진 개발자 커뮤니티에서 만들었습니다. 작은 여우 로고가 너무 귀여워서 거부할 수 없습니다.

그런 다음 아무도 요청하지 않았지만 어쨌든 얻은 브라우저인 Microsoft Edge가 있습니다. 예전에는 Internet Explorer라고 불렸지만 Microsoft는 아무도 월요일 아침에 나무늘보보다 느린 브라우저를 사용하고 싶어하지 않는다는 것을 깨달았습니다. 그래서 그들은 그것을 리브랜딩하고 페이스리프트를 했습니다. 나쁘지는 않지만 좋지도 않습니다. 항상 모든 사람을 기쁘게 하려고 노력하지만 부족해지는 브라우저의 중간 자식과 같습니다.

마지막으로 Safari가 있습니다. 달콤하고 달콤한 Cupertino의 장점을 충분히 얻을 수 없는 Apple 광신자를 위한 브라우저입니다. 빠르고 날렵하며 Mac 또는 iPhone에서 인터넷을 탐색하는 데 필요한 모든 기능을 갖추고 있습니다. 그러나 현실을 직시하자. 사람들이 Safari를 사용하는 유일한 이유는 Chrome을 다운로드하기에 너무 게으르기 때문입니다.

자, 여러분, 가장 인기 있는 웹 브라우저에 대한 간략한 설명이 있습니다. 당신이 크롬 광신자이든 파이어폭스 지지자이든, 우리는 모두 한 가지에 동의할 수 있습니다. 우리는 신뢰할 수 있는 브라우저 없이는 길을 잃을 것입니다. 결국, 하루 종일 직장에서 미루는 것이 또 어떻게 가능하겠습니까?
