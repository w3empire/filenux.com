브라우저의 힘을 잠금 해제: Firefox 팁, 요령 및 숨겨진 기능에 대한 전문가 가이드
전문가 가이드로 Firefox를 마스터하세요. 탐색, 개인 정보 보호, 검색 최적화 및 사용자 경험을 향상시키는 팁, 요령 및 숨겨진 기능을 발견하십시오.

개발자: Bruno Souza
업데이트됨: 2023년 05월 14일

Mozilla Firefox는 오늘날 가장 널리 사용되는 웹 브라우저 중 하나이며 속도, 보안 및 사용자 정의 기능으로 유명합니다. 사용자 친화적이지만 Firefox를 진정으로 마스터하고 브라우징 경험을 최적화하려면 몇 가지 필수 팁과 요령을 배울 가치가 있습니다. 이 가이드는 Firefox를 최대한 활용하는 데 도움이 되는 숨겨진 기능 중 일부를 탐색합니다.

인터페이스 개인화

Firefox의 가장 큰 강점 중 하나는 사용자 지정 기능입니다. 브라우징 습관에 맞게 인터페이스를 개인화할 수 있습니다.

고정 및 재정렬 탭

항상 같은 웹 사이트를 여는 경우 쉽게 액세스할 수 있도록 이러한 탭을 고정할 수 있습니다. 고정하려는 탭을 마우스 오른쪽 버튼으로 클릭하고 “탭 고정"을 선택합니다. 고정된 탭은 탭 표시줄의 왼쪽으로 이동하고 사이트 아이콘만 표시하도록 축소됩니다. 탭을 재정렬하려면 탭을 클릭한 상태에서 원하는 위치로 드래그하면 됩니다.

도구 모음 사용자 지정

Firefox를 사용하면 도구 모음에서 항목을 추가, 제거 및 재정렬할 수 있습니다. 메뉴 버튼(오른쪽 상단 모서리에 있는 3개의 가로줄)을 클릭한 다음 “사용자 지정"을 클릭합니다. 원하는 항목을 도구 모음으로 끌어다 놓을 수 있습니다. 완료되면 “완료"를 클릭하십시오.

검색 최적화

Firefox는 온라인 검색 속도를 높이는 데 도움이 되는 다양한 도구를 제공합니다.

검색 시 주소 표시줄 사용

검색 엔진의 홈페이지로 이동하는 대신 Firefox 주소 표시줄에 직접 검색어를 입력할 수 있습니다. 자동으로 기본 검색 엔진을 사용합니다.

검색 엔진 바로 가기

즉시 검색 엔진을 전환할 수도 있습니다. 주소 표시줄에 검색어를 입력한 다음 돋보기 아이콘을 클릭하여 사용 가능한 검색 엔진의 드롭다운 메뉴를 표시합니다.

키워드 북마크

Firefox에서는 책갈피에 키워드를 할당할 수 있습니다. 이렇게 하려면 북마크를 마우스 오른쪽 버튼으로 클릭하고 ‘속성’을 선택한 다음 ‘키워드’ 필드를 입력합니다. 이렇게 하면 주소 표시줄에 키워드를 입력하기만 하면 북마크된 페이지에 액세스할 수 있습니다.

프라이버시 강화

Firefox는 사용자 개인 정보 보호에 대한 강한 의지로 유명합니다.

프라이빗 브라우징

디지털 발자국을 남기고 싶지 않은 상황에서는 Firefox의 비공개 브라우징 모드를 사용하세요. 메뉴 버튼을 클릭하고 “새 사생활 보호 창"을 선택합니다. 이 모드에서 Firefox는 검색 기록, 검색 기록 또는 쿠키를 저장하지 않습니다.

향상된 추적 보호

Firefox의 향상된 추적 보호 기능은 알려진 많은 타사 추적기를 자동으로 차단합니다. 이러한 설정을 사용자 지정하려면 주소 표시줄 왼쪽에 있는 방패 아이콘을 클릭하십시오. “Standard”, “Strict” 또는 “Custom” 모드 중에서 선택할 수 있습니다.

키보드 단축키 활용

키보드 단축키는 브라우징 속도를 크게 향상시킬 수 있습니다. 다음은 가장 유용한 몇 가지입니다.

윈도우/리눅스:

  • Ctrl+T: 새 탭을 엽니다.
  • Ctrl+W: 현재 탭을 닫습니다.
  • Ctrl+Shift+T: 마지막으로 닫은 탭을 다시 엽니다.
  • Ctrl+L: 주소 표시줄을 강조 표시합니다.
  • Ctrl+R: 현재 페이지를 새로 고칩니다.
  • Ctrl+D: 현재 페이지를 북마크합니다.

맥 OS:

  • Command+T: 새 탭을 엽니다.
  • Command+W: 현재 탭을 닫습니다.
  • Command+Shift+T: 마지막으로 닫은 탭을 다시 엽니다.
  • Command+L: 주소 표시줄을 강조 표시합니다.
  • Command+R: 현재 페이지를 새로 고칩니다.
  • Command+D: 현재 페이지를 북마크합니다.

Firefox 동기화 사용

여러 기기를 사용하는 경우 Firefox Sync는 북마크, 방문 기록, 탭 및 비밀번호를 동기화 상태로 유지할 수 있습니다. 설정하려면 메뉴 버튼을 클릭한 다음 ‘Firefox에 로그인’을 클릭하세요. 계정을 생성하고 로그인한 후 “동기화 설정"에서 동기화할 정보를 선택할 수 있습니다.

애드온 및 확장

Firefox의 부가 기능과 확장 기능은 브라우징 경험을 크게 향상시킬 수 있습니다. 액세스하려면 메뉴 버튼을 클릭한 다음 “추가 기능"을 클릭하십시오. 개인 정보 보호, 생산성, 테마 등을 위한 방대한 확장 라이브러리를 찾을 수 있습니다. 광범위한 권한을 가질 수 있으므로 신뢰할 수 있는 소스의 확장 프로그램만 설치해야 합니다.

권장 애드온

다음은 Firefox에서 널리 사용되고 높은 평가를 받는 몇 가지 추가 기능 입니다.

  • uBlock Origin: 메모리와 CPU에 효율적인 강력한 광고 차단기입니다.
  • Privacy Badger: 보이지 않는 트래커를 차단하는 방법을 자동으로 학습합니다.
  • HTTPS by default: 웹 사이트를 안전하지 않은 “http"에서 보안된 “https"로 자동 전환합니다.

Firefox 개발자 도구 활용

Firefox는 웹 개발에 관심이 있거나 자신이 좋아하는 웹사이트의 내부 작업에 대해 궁금한 사람들을 위해 강력한 개발자 도구 세트를 제공합니다.

  • 검사기: 모든 웹페이지의 기본 HTML 및 CSS를 엿볼 수 있습니다.
  • 콘솔: JavaScript, 네트워크 요청, 보안 오류 또는 경고와 같은 웹 페이지와 관련된 정보를 기록합니다.
  • 디버거: JavaScript에서 문제를 찾고 수정하는 데 도움이 됩니다.
  • 네트워크: 브라우저에서 직접 네트워크 요청을 모니터링하고 디버깅합니다.

이러한 도구에 액세스하려면 웹 페이지를 마우스 오른쪽 버튼으로 클릭하고 “요소 검사"를 선택하십시오.

포켓 통합

Pocket은 나중에 볼 기사, 비디오 및 페이지를 저장할 수 있는 Firefox에 통합된 도구입니다. 페이지를 저장하려면 주소 표시줄의 주머니 아이콘을 클릭하기만 하면 됩니다. 저장된 페이지에 액세스하려면 라이브러리 버튼(책장 아이콘)을 클릭한 다음 “포켓 목록 보기"를 클릭하십시오.

결론

오랫동안 Firefox를 사용했든 방금 전환했든 이 팁과 요령은 이 강력한 웹 브라우저를 최대한 활용하는 데 도움이 될 수 있습니다. 인터페이스 사용자 정의에서 개인 정보 보호 강화, 검색 속도 향상, 키보드 단축키 활용 및 추가 기능 활용에 이르기까지 Firefox는 브라우징 경험을 최적화하기 위한 다양한 도구를 제공합니다. 이를 마스터하면 비교할 수 없는 속도, 효율성 및 보안으로 웹을 탐색할 수 있습니다.

Mozilla Firefox 경험을 더욱 향상시키기 위한 추가 팁

1. 리더뷰

산만하지 않고 기사를 즐기고 싶다면 Firefox의 리더 뷰는 축복입니다. 활성화할 페이지에 있을 때 주소 표시줄 오른쪽에 있는 “책” 아이콘을 클릭하십시오. Reader View는 텍스트와 필수 이미지만 남기고 광고, 사이드바 및 기타 불필요한 요소를 제거합니다. 원하는 대로 텍스트 크기, 너비 및 테마를 조정할 수도 있습니다.

2. 스크린샷 도구

Firefox에는 전체 페이지, 선택한 영역 또는 페이지의 특정 요소를 캡처할 수 있는 내장 스크린샷 도구가 있습니다. 주소 표시줄 오른쪽에 있는 세 개의 점을 클릭하고 “스크린샷 찍기"를 선택합니다. 스크린샷을 다운로드하거나 Firefox 계정에 저장하여 다른 장치에서 액세스할 수 있습니다.

3. 컨테이너 탭

Firefox를 사용하면 온라인 활동을 다른 “컨테이너"로 분리할 수 있습니다. 예를 들어 개인 브라우징을 업무와 분리하거나 동일한 브라우저에서 여러 소셜 미디어 계정을 열어 두는 데 유용할 수 있습니다. 이 기능에 액세스하려면 “Firefox 다중 계정 컨테이너” 확장 프로그램을 다운로드해야 합니다.

4. 삭제 버튼 사용

최근 검색 기록을 빠르게 지우려면 삭제 버튼을 사용하세요. 이를 통해 단 한 번의 클릭으로 지난 5분, 2시간 또는 24시간의 인터넷 사용 기록을 삭제할 수 있습니다. 도구 모음에 삭제 버튼을 추가하려면 메뉴 버튼(오른쪽 상단 모서리에 있는 3개의 가로선)을 클릭한 다음 “사용자 지정"을 클릭합니다. 삭제 버튼을 도구 모음으로 끌어다 놓을 수 있습니다.

5. 다크 모드 활성화

특히 밤에 검색할 때 더 어두운 색 구성표를 선호하는 경우 Firefox의 어두운 모드를 활성화할 수 있습니다. 메뉴 버튼으로 이동하여 ‘사용자 지정’을 클릭하면 화면 하단에 ‘테마’가 표시됩니다. 그것을 클릭하고 “어두움"을 선택하십시오.

이러한 추가 팁을 사용하면 Firefox를 훨씬 더 효율적이고 효과적으로 탐색할 수 있습니다.
