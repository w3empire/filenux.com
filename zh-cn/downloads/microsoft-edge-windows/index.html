安全、免费、快捷

Microsoft Edge 是 Microsoft 开发的 Web 浏览器，于 2015 年首次发布。它旨在成为 Internet Explorer 的现代且更快的替代品，Internet Explorer 是以前版本 Windows 上的默认 Web 浏览器。

Microsoft Edge 的一个突出特点是它的速度。它使用一种名为 EdgeHTML 的新呈现引擎，该引擎针对现代网络标准进行了优化，这使其能够比其他浏览器更快、更高效地加载页面。Edge 也有一个干净简单的界面，具有阅读模式等功能，可以消除干扰，让你更容易专注于你正在阅读的内容。

除了其性能和设计，Microsoft Edge 还提供一系列安全和隐私功能，以在用户浏览 Web 时保护他们。其中包括阻止弹出窗口和不需要的 cookie 的选项，以及可以生成强密码并将其保存以备后用的内置密码管理器。

Microsoft Edge 的另一个显着特点是它与微软其他产品和服务的集成。例如，您可以使用浏览器访问您的 OneDrive 帐户或使用注释工具标记网页并与其他人共享。Microsoft Edge 还支持“扩展程序”，这些程序是为浏览器添加额外功能的小程序。

总而言之，Microsoft Edge 是一款快速、安全且功能丰富的 Web 浏览器，非常适合个人和专业用途。无论您是休闲浏览网页还是处理重要文档，Edge 都拥有完成工作所需的工具和功能。

特点和亮点

用户界面： Microsoft Edge 带有一个用户友好的界面，干净、现代且易于导航。它采用极简主义设计，使用户可以轻松地将注意力集中在他们正在查看的内容上。

选项卡管理： Edge 提供了出色的选项卡管理功能。您可以使用键盘快捷键、鼠标单击或触摸手势轻松打开、关闭和在选项卡之间切换。它还允许您预留当前未使用但稍后想访问的选项卡。

阅读视图： Microsoft Edge 提供了一个“阅读视图”选项，可以去除网页中的广告和干扰，只留下文章的主要文本和图像。此功能使阅读网页变得更容易，并且不会分散注意力。

Cortana 集成： Cortana 是微软的虚拟助手，Edge 与 Cortana 集成，为用户提供额外的功能。例如，您可以要求 Cortana 帮助您查找网站或获取有关某个主题的信息。

扩展： Edge 支持扩展，允许用户向浏览器添加额外的功能。有广告拦截、密码管理、社交媒体等扩展。

PDF 支持： Edge 可以查看和注释 PDF 文件，而无需任何其他软件。此功能对于需要经常处理 PDF 文档的用户特别有用。

安全性： Microsoft Edge 具有多项安全功能，可帮助保护用户免受恶意网站和攻击。它具有内置的网络钓鱼过滤器、恶意软件保护和沙盒技术，可将浏览器与操作系统隔离开来。

用户界面

Microsoft Edge 用户界面设计简洁、现代且用户友好，重点是简单性和功能性。用户界面的关键元素包括：

地址栏：地址栏位于浏览器窗口的顶部，允许用户输入 URL 或搜索查询。Edge 使用 Bing 作为默认搜索引擎，但用户可以在设置中将其更改为自己喜欢的搜索引擎。

选项卡：在地址栏上方，用户可以使用选项卡式界面管理多个打开的网页。每个选项卡都显示页面的标题，如果可用，还显示一个便于识别的网站图标。

新标签页：打开新标签页时，用户会看到一个可自定义的起始页，其中显示搜索栏、经常访问的网站以及个性化内容，例如新闻和天气更新。

导航按钮：位于地址栏的左侧，用户可以找到标准的后退、前进和刷新按钮，以便于导航。

菜单：单击窗口右上角的三点图标（省略号）可打开主菜单，从中可以访问各种浏览器功能和设置，例如书签、历史记录、扩展程序、隐私和安全选项等.

集合： Microsoft Edge 的一项独特功能，集合允许用户在专用侧边栏中收集和组织 Web 内容，例如链接、图像和文本。可以通过单击地址栏右侧的收藏图标来访问此功能。

收藏夹和阅读列表：用户可以保存自己喜欢的网站以便以后轻松访问，阅读列表功能可以让他们保存文章或网页以供以后阅读或离线阅读。

扩展： Microsoft Edge 支持范围广泛的扩展，包括来自 Chrome 网上应用店的扩展，它们可以增强浏览器功能并自定义用户体验。

辅助功能： Edge 提供各种辅助功能，例如沉浸式阅读器模式，可简化网页布局并提供文本到语音转换功能，以及高对比度主题以帮助有视觉障碍的用户。

Microsoft Edge 的用户界面旨在提供无缝且高效的浏览体验，所有体验级别的用户都可以轻松浏览。

选项卡管理

微软开发的网络浏览器 Microsoft Edge 引入了多项旨在改善用户体验的功能，标签管理就是其中之一。Microsoft Edge 中的选项卡管理旨在帮助用户有效地组织、导航和处理多个打开的选项卡。

Microsoft Edge 选项卡管理功能的一些关键方面包括：

垂直选项卡：此功能允许用户将打开的选项卡从传统的水平布局移动到窗口侧面的垂直堆栈。这提供了一个更好的打开选项卡概览，更容易找到它们并在它们之间切换。要启用垂直标签，用户只需单击浏览器窗口左上角的“垂直标签”按钮即可。

选项卡组：选项卡组使用户能够在单个组下对相似的选项卡进行分类和标记。这有助于保持浏览器井井有条，并使其更容易在相关选项卡之间导航。要创建选项卡组，用户可以右键单击选项卡并选择“将选项卡添加到新组”或“将选项卡添加到现有组”。

选项卡预览：此功能可在将鼠标悬停在选项卡上时提供选项卡内容的小预览。这可以帮助用户快速识别选项卡的内容，而无需切换到该选项卡。

睡眠标签： Microsoft Edge 引入了一项名为“睡眠标签”的功能，以帮助节省系统资源。当某个选项卡在一段时间内未被使用时，它会自动进入休眠模式，释放系统资源。用户可以自定义标签进入休眠前的时间段，点击时休眠标签会自动重新加载。

固定标签：用户可以固定经常访问的网站或重要的标签，使它们在标签行的开头永久可见。这样可以更轻松地访问最重要或最常用的选项卡。

键盘快捷键： Microsoft Edge 支持多种键盘快捷键用于选项卡管理，例如 Ctrl+T 用于打开新选项卡，Ctrl+W 用于关闭当前选项卡，Ctrl+Tab 用于在打开的选项卡之间切换。

Microsoft Edge 中的这些选项卡管理功能显着改善了整体浏览体验，使同时管理多个选项卡的用户更有效率和更有条理。

阅读视图

Microsoft Edge 的阅读视图功能是一个方便的工具，旨在改善用户的在线阅读体验。它简化了网页的布局，消除了干扰，例如广告、弹出窗口和其他非必要元素，从而更容易将注意力集中在网页的主要内容上。

以下是 Microsoft Edge 阅读视图功能的一些关键方面：

更简洁的布局：通过移除广告、侧边栏和其他杂物，阅读视图呈现了一个干净、整洁的界面，可帮助用户专注于他们正在阅读的文本。这在阅读长文章或密集内容时特别有用。

自定义：用户可以通过选择不同的字体样式、文本大小和背景颜色来自定义阅读视图的外观。这允许个性化的阅读体验，以满足个人喜好和可访问性需求。

文本转语音：该功能提供称为“大声朗读”的内置文本转语音功能，可以大声朗读内容。这对有视觉障碍的用户或喜欢听内容而不是阅读内容的用户很有帮助。

离线阅读：阅读视图允许用户保存网页以供离线阅读，即使在没有互联网连接的情况下也可以方便地访问内容。

轻松访问：使用地址栏中的书籍图标或按键盘快捷键 Ctrl+Shift+R，可以轻松打开和关闭阅读视图功能。这使得根据需要在标准视图和简化视图之间切换变得简单。

Microsoft Edge 的阅读视图是一种非常有用的工具，可在阅读在线内容时增强用户体验。通过提供自定义选项和辅助功能，它可以满足广泛的用户需求和偏好。

Cortana 集成

Microsoft Edge 集成了该公司的虚拟助手 Cortana，以增强用户体验并提供更加无缝的浏览体验。这种集成称为 Cortana 集成功能，允许用户直接在 Edge 浏览器中与虚拟助手进行交互。虽然 Cortana 最初是随 Windows 10 引入的，但它也可用于其他平台和设备。

Microsoft Edge 中的 Cortana 集成提供了多项优势和功能，包括：

语音搜索：用户可以直接从地址栏使用 Cortana 进行语音搜索。此功能使用户无需手动输入查询即可搜索信息。

上下文信息： Cortana 可以为网页上选定的文本提供上下文信息。通过突出显示文本和单击鼠标右键，用户无需离开当前页面即可访问 Cortana 的帮助并获取相关信息或定义。

个性化建议： Cortana 可以根据用户的浏览历史、偏好和兴趣提供个性化建议。这可以帮助用户发现新内容并在搜索特定信息时节省时间。

提醒和通知： Cortana 可以直接通过 Microsoft Edge 浏览器发送提醒和通知。用户无需打开单独的应用程序或窗口即可设置提醒或接收重要提醒。

与其他 Microsoft 服务集成： Cortana 可与 Outlook、OneNote 和 To-Do 等其他 Microsoft 服务无缝协作，允许用户直接从浏览器管理任务、约会和笔记。

跨设备同步： Cortana 可以跨设备同步信息，因此用户可以从他们的智能手机、平板电脑或 PC 访问提醒、笔记和其他数据。

值得注意的是，微软已经将其重点从作为独立虚拟助手的 Cortana 转移到将其集成到微软生态系统中更具体的生产力场景中。因此，虚拟助手的可用性和功能可能因地区和设备而异。

然而，Microsoft Edge 中的 Cortana 集成功能仍然为用户提供了一系列好处和便利，增强了他们的整体浏览体验。

支持扩展

扩展程序是可以修改和增强浏览器功能的小型软件程序，允许用户根据自己的需要和偏好定制浏览体验。

Microsoft Edge 对扩展的支持提供了几个好处：

广泛的扩展：由于 Microsoft Edge 基于 Chromium，它可以支持 Chrome 网上应用店中提供的绝大多数 Chrome 扩展。这使用户可以轻松找到并安装适合他们需要的扩展，因为已经有一个庞大且已建立的选项库可供选择。

易于安装：在 Microsoft Edge 上安装扩展非常简单。用户可以访问 Microsoft Edge Add-ons 网站或 Chrome Web Store 来浏览和安装扩展。找到所需的扩展程序后，用户可以单击“添加到 Chrome”或“获取”按钮进行安装，扩展程序将添加到他们的浏览器中。

扩展管理： Microsoft Edge 提供了一个易于使用的界面来管理已安装的扩展。用户可以通过单击浏览器右上角的三点菜单图标并选择“扩展”来访问扩展页面。从那里，用户可以启用、禁用或删除扩展，以及访问单个扩展的设置。

安全和隐私： Microsoft 致力于确保其用户的安全和隐私，这延伸到对扩展的支持。Microsoft Edge 对提交到其附加组件网站的扩展程序有一个审查流程，以确保它们在提供给用户之前满足某些标准。此外，用户可以控制授予每个扩展的权限，限制他们可以访问的数据。

开发人员支持： Microsoft 为有兴趣为 Microsoft Edge 创建扩展的开发人员提供资源和文档。开发人员可以使用 Microsoft Edge Extension Toolkit 转换现有的 Chrome 扩展或从头开始创建新的。这种支持鼓励扩展生态系统的发展，并确保用户可以访问各种工具和功能。

PDF 支持

Microsoft Edge 的 PDF 功能为用户提供了一种无需第三方软件即可查看、导航、注释和管理 PDF 文件的便捷方式。以下是 Microsoft Edge 提供的一些用于处理 PDF 文档的主要功能：

查看 PDF： Microsoft Edge 使用户能够直接在浏览器中打开和查看 PDF 文件。可以从本地驱动器或通过 URL 访问这些文件。

导航：浏览器提供直观的导航工具，例如用于查看文档缩略图的侧边栏、目录、书签，以及通过输入页码跳转到特定页面的能力。

缩放和视图选项： Edge 提供一系列缩放级别和视图选项，例如单页、连续滚动和两页视图，以增强可读性和适应用户偏好。

批注：用户可以在 PDF 文本中添加注释、高亮、下划线和删除线，方便在阅读文档时进行协作或做笔记。此外，用户可以在 PDF 中绘制或添加形状和文本框。

文本选择和复制： Microsoft Edge 允许用户从 PDF 文件中选择和复制文本，然后可以将其粘贴到其他应用程序中。

搜索：浏览器包含一个内置的搜索功能，使用户能够在 PDF 文档中查找特定的文本或短语。

填写和签署表单： Edge 支持填写和签署 PDF 表单，允许用户直接在浏览器内完成和提交各种类型的表单。

辅助功能： Microsoft Edge 为屏幕阅读器等辅助技术提供内置支持，并提供启用高对比度模式或大声朗读的选项，以提高视障用户的辅助功能。

保存和打印：用户可以轻松地将 PDF 文件保存到本地驱动器或直接从浏览器打印。

与其他 Microsoft 服务集成： Edge 与其他 Microsoft 服务（例如 OneDrive、SharePoint 和 Outlook）无缝集成，从而可以轻松地跨各种平台共享、访问和管理 PDF 文件。

虽然 Microsoft Edge 提供了一组用于处理 PDF 文档的强大功能，但它可能不包括专用 PDF 软件或第三方浏览器扩展提供的所有高级功能。但对于大多数用户来说，Edge 内置的 PDF 支持对于日常任务和文档管理来说应该绰绰有余。

安全功能

Microsoft Edge 非常重视安全性，以确保为其用户提供安全的浏览体验。Microsoft Edge 中包含的一些关键安全功能包括：

SmartScreen 筛选器： SmartScreen 是 Microsoft 开发的一项功能，可帮助保护用户免受网络钓鱼攻击、恶意软件和其他恶意软件的侵害。它根据不断更新的已报告恶意网站和下载列表检查网站和文件的 URL，在用户即将访问具有潜在危险的网站或下载具有潜在危害的文件时提醒用户。

沙盒： Edge 使用多进程架构，将 Web 内容隔离到沙盒进程中。这意味着，如果恶意网站设法利用浏览器中的漏洞，损害是有限的，并且攻击者无法访问用户的个人数据或其他系统资源。

Microsoft Defender 集成： Microsoft Edge 内置了与 Microsoft Defender 的集成，Microsoft Defender 是该公司的防病毒和反恶意软件解决方案。此集成提供对通过浏览器访问的下载和文件的实时扫描，以帮助检测和阻止可能有害的文件。

跟踪预防： Microsoft Edge 提供跟踪预防选项，让用户更好地控制他们的在线隐私。用户可以选择三种不同级别的跟踪预防：基本、平衡或严格。每个级别都会阻止不同类别的跟踪器，严格的设置可提供最高级别的隐私保护。

InPrivate Browsing：与其他浏览器的隐私浏览模式类似，Edge 的 InPrivate Browsing 允许用户在没有浏览历史记录、临时互联网文件或 cookie 存储在他们的设备上的情况下浏览网页。

安全更新： Microsoft Edge 定期更新安全补丁和改进。这些更新通过 Windows Update 安全交付，以帮助确保用户免受最新威胁的侵害。

密码监控器：此功能可在用户保存的密码涉及已知数据泄露时提醒他们，从而帮助用户保护其在线帐户的安全。然后提示用户更改受影响的密码以维护帐户安全。

默认为 HTTPS： Microsoft Edge 会在可用时自动将与网站的连接从 HTTP 升级到 HTTPS，从而提供更安全和加密的连接来保护用户数据。

Microsoft Edge 基于Chromium，这是一个开源浏览器项目，也是 Google Chrome 的基础。2018 年 12 月，微软宣布将使用 Chromium 引擎重建 Edge，以提高兼容性和性能。新版 Edge 于 2020 年 1 月发布，现在基于与 Google Chrome 相同的核心技术。

虽然 Microsoft Edge 和Google Chrome在 Chromium 中共享一个共同的基础，但它们仍然是具有自己独特功能和设计的截然不同的浏览器。例如，Edge 包含了 Chrome 中没有的阅读模式和注释工具，它还集成了微软的其他产品和服务。但是，这两个浏览器能够运行相同的扩展程序和 Web 应用程序，并且它们还具有相似的性能和安全功能。

浏览器之战：对顶级互联网导航器的机智评论

我们赖以探索广阔无垠的互联网领域的可靠工具。无论您是搜索猫咪视频还是购买最新的指尖陀螺，网络浏览器都是必不可少的。但老实说，它们并非生而平等。有些流畅而快速，而另一些则像星期天早上的蜗牛一样笨重而缓慢。

首先，我们有全能的谷歌浏览器。它是山丘之王，头号狗，浏览器中的碧昂丝。Chrome 浏览器运行速度快如闪电，并且具备修复互联网所需的所有功能。此外，它是唯一一款在您的互联网中断时弹出恐龙游戏的浏览器。你的浏览器能做到吗？我不这么认为。

接下来，我们有Mozilla Firefox。这就像总是试图赶上 Chrome 的好斗的失败者。当然，它没有那么快，但它有心。Firefox 是开源的，这意味着它是由开发人员社区创建的，这些开发人员热衷于制作一款免费且可供所有人访问的浏览器。说实话，小狐狸标志实在是太可爱了，让人无法抗拒。

然后是 Microsoft Edge，这是一款无人问津但我们还是得到了的浏览器。它曾经被称为 Internet Explorer，但微软意识到没有人愿意在周一早上使用比树懒还慢的浏览器。因此，他们重新命名并进行了整容。这还不错，但也不是很好。它就像浏览器中的老二，总是试图取悦所有人，却功亏一篑。

最后但同样重要的是，我们有 Safari。它是苹果狂热者的浏览器，他们只是无法获得足够的甜美，甜美的库比蒂诺善良。它快速、时尚，并且具备在 Mac 或 iPhone 上浏览互联网所需的所有功能。但说实话：人们使用 Safari 的唯一原因是他们懒得下载 Chrome。

所以，伙计们，你有它，一个最流行的网络浏览器的快速概述。无论您是 Chrome 狂热者还是 Firefox 支持者，我们都同意一件事：没有我们值得信赖的浏览器，我们就会迷路。毕竟，我们怎么能整天拖延工作呢？
