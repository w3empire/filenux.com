Waterfox 是基于Mozilla Firefox代码库的开源 Web 浏览器，适用于 Windows、macOS 和 Linux 。它由当时 16 岁的学生 Alex Kontos 于 2011 年创建，目标是创建一个针对 64 位系统优化且更注重隐私的 Firefox 版本。

Waterfox Classic 是 Waterfox 网络浏览器的一个版本，它基于早期版本的 Firefox 代码库（特别是版本 56），旨在保持与旧版 Firefox 附加组件和扩展的兼容性。Waterfox Classic 提供 32 位和 64 位版本，可从 Waterfox 官方网站下载。

与较新版本的 Waterfox 不同，Waterfox Classic 不再积极开发或维护，这意味着它可能不会收到安全更新或错误修复。但是，对于依赖旧版 Firefox 附加组件或更喜欢早期版本 Firefox 的界面和功能的用户来说，它仍然是一个有用的选择。

请务必注意，不建议需要最新安全更新和现代 Web 功能的用户使用 Waterfox Classic，应谨慎使用。

64 位架构

Waterfox 的一个显着方面是它能够使用 64 位架构。虽然 Firefox 支持 64 位系统已有一段时间，但 Waterfox 是专为 64 位系统设计的首批浏览器版本之一。这意味着它能够利用这些系统的额外资源和功能，从而获得更快的性能和更高效的浏览体验。

隐私

除了 64 位支持外，Waterfox 还专注于隐私。它包括许多以隐私为中心的功能，例如禁用遥测和数据收集的能力，以及对增强隐私的附加组件（如 uBlock Origin 和 Privacy Badger）的支持。Waterfox 在构建时也考虑到了注重隐私的用户，具有第一方隔离等功能，有助于防止跨站点跟踪，并支持 TOR 网络。

兼容性

Waterfox 的另一个关键特性是它与旧版附加组件的兼容性。许多较旧的 Firefox 附加组件不再与较新版本的浏览器兼容，但 Waterfox 旨在能够运行这些旧版附加组件。这使其成为依赖这些附加组件并希望继续使用它们的用户的热门选择。

支持扩展

除了其核心特性外，Waterfox 还包括许多与 Firefox 相同的特性和功能，包括对扩展的支持、书签管理器、拼写检查器和密码管理器。它还包括对范围广泛的 Web 标准和技术的支持，包括 HTML5、CSS3 和 JavaScript。

Waterfox 社区

Waterfox 可在 FileNUX 网站或官方 Waterfox 网站上下载，并根据 GNU 通用公共许可证 (GPL) 分发。它由一小群志愿者开发和维护，并得到更广泛的 Waterfox 社区的支持。

经常问的问题

什么是 Waterfox Classic Web 浏览器？

Waterfox 是一款基于 Mozilla Firefox 的免费开源 Web 浏览器。它于 2011 年由 Alex Kontos 首次发布，旨在比 Firefox 更快、更注重隐私。

Waterfox Web 浏览器是否可用于多个操作系统？

是的，Waterfox 适用于 Windows、Mac OS 和 Linux 操作系统。

Waterfox 与 Firefox 有何不同？

Waterfox 本质上是 Firefox 的一个分支，这意味着它使用与 Firefox 相同的代码库，但有一些修改。与 Firefox 相比，Waterfox 专注于更快的速度和增强的隐私功能。

Waterfox 提供哪些隐私功能？

Waterfox 提供了许多隐私功能，例如默认情况下阻止跟踪器和广告、禁用遥测和数据收集以及支持 Tor 浏览器附加组件以增强匿名性。

Waterfox 网络浏览器安全吗？

Waterfox 将安全放在首位并提供定期安全更新。此外，其隐私功能有助于保护用户的个人信息和浏览活动。

Waterfox 有哪些附加组件和扩展？

Waterfox 支持大多数 Firefox 附加组件和扩展，以及它自己的专有附加组件。

如何将我的书签和设置从 Firefox 导入到 Waterfox？

Waterfox 包含一个导入工具，可让您轻松地从 Firefox 传输书签、历史记录、密码和其他设置。

Waterfox 是否支持跨多个设备同步？

是的，Waterfox Sync 允许您在多个设备上同步您的书签、历史记录、密码和打开的标签页。

Waterfox 网络浏览器可以免费使用吗？

是的，Waterfox 是完全免费的开源软件。

我可以为 Waterfox 网络浏览器的开发做出贡献吗？

是的，Waterfox 是一个开源项目，欢迎来自社区的贡献。您可以通过报告错误、开发附加组件甚至贡献源代码来为 Waterfox 做出贡献。

Waterfox 是否支持 Windows 32 位系统？

是的，Waterfox 网络浏览器同时支持 32 位和 64 位版本的 Windows 操作系统。当您从官方网站下载 Waterfox 安装程序时，您可以根据您的操作系统选择 32 位或 64 位版本。但是请注意，从 2021.02 版开始，Waterfox 仅支持 64 位版本的 Windows。如果您有较旧的 32 位 Windows 系统，您仍然可以下载和使用支持 32 位系统的较旧版本的 Waterfox。

如何安装 Waterfox Web 浏览器（逐步）

以下是有关如何在 Windows 上安装 Waterfox Web 浏览器的分步指南：

  1. 单击此页面顶部的“下载”按钮开始下载 Waterfox Classic 安装程序。
  2. 下载完成后，双击下载的文件开始安装过程。
  3. 在 Waterfox 设置窗口中，选择您的首选语言并单击“确定”。
  4. 阅读并接受 Waterfox 许可协议，然后单击“下一步”。
  5. 选择要安装 Waterfox 的目标文件夹，或使用默认文件夹，然后单击“下一步”。
  6. 选择您要安装的组件，例如 Waterfox 浏览器和 Waterfox Sync，然后单击“下一步”。
  7. 选择要创建 Waterfox 快捷方式的开始菜单文件夹，或使用默认文件夹，然后单击“下一步”。
  8. 选择是否要为 Waterfox 创建桌面快捷方式，然后单击“下一步”。
  9. 选择是否要将 Waterfox 设置为默认浏览器，然后单击“安装”。
  10. 等待安装过程完成，然后单击“完成”。

Waterfox 现已安装在您的 Windows 计算机上，您可以通过双击桌面快捷方式或在“开始”菜单中搜索来打开它。

Waterfox Web 浏览器的系统要求

Windows 上的 Waterfox Classic Web 浏览器的系统要求是：

  • 操作系统： Windows 7、8、8.1、10（32 位或 64 位）或更高版本的 Windows。
  • 处理器：支持 SSE2 的 Intel Pentium 4 处理器或更高版本。
  • 内存 (RAM)： 2 GB 或更多。
  • 硬盘空间：至少 200 MB 可用空间。
  • 显卡：支持 DirectX-9 的显卡，带有 WDDM 驱动程序。
  • 显示器： 800x600 或更高分辨率。

请注意，这些是 Waterfox 的最低系统要求，浏览器可能会在更强大的硬件上运行得更好。此外，某些 Waterfox 功能（例如视频硬件加速）需要特定的硬件和软件配置。
