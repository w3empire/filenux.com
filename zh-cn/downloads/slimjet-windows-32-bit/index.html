经常问的问题

什么是 Slimjet Web 浏览器？

Slimjet 是一款基于 Chromium 开源项目的免费开源 Web 浏览器。它由 FlashPeak Inc. 开发并于 2015 年发布。

Slimjet 网络浏览器的系统要求是什么？

Slimjet 支持 Windows、Mac 和 Linux 操作系统。Slimjet 的最低系统要求是 Windows 7、Mac OS X 10.9 或任何 64 位版本的 Linux。

Slimjet 网络浏览器有哪些功能？

Slimjet 具有多项功能，包括广告拦截器、下载管理器、Facebook 集成、即时照片上传和可自定义的工具栏。

Slimjet Web 浏览器安全吗？

是的，Slimjet 是安全的。它具有网络钓鱼和恶意软件防护等内置功能，可确保用户在浏览时的安全。

我可以将书签导入 Slimjet 网络浏览器吗？

是的，您可以轻松地将书签从其他网络浏览器（如 Chrome、Firefox 和 Internet Explorer）导入到 Slimjet。

Slimjet Web 浏览器是否支持扩展？

是的，Slimjet 支持扩展，用户可以从 Chrome 网上应用店下载扩展。

Slimjet Web 浏览器是免费的吗？

是的，Slimjet 是一款免费的网络浏览器。但是，有一个高级版本可供需要额外功能的用户使用。

Slimjet Web Browser 有一个名为“Slimjet VIP”的高级版本。它是付费版本，提供额外的功能和服务，例如广告拦截、反跟踪和 VPN 服务。高级版还提供优先技术支持和更新。

用户可以选择购买一次性许可证或订阅 Slimjet VIP。定价可能因许可证类型和订阅期限而异。不过，标准版的 Slimjet Web Browser 是免费使用的，用户仍然可以享受各种实用的功能，而无需支付任何费用。

如何更新 Slimjet Web 浏览器？

默认情况下，Slimjet 会自动更新。但是，您也可以在设置菜单中手动检查更新。

我可以在移动设备上使用 Slimjet 网络浏览器吗？

不，Slimjet 目前不适用于移动设备。它仅支持桌面操作系统。
