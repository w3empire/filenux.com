Yandex Browser 是由俄罗斯科技公司 Yandex 开发的免费网络浏览器。它于 2012 年首次发布，适用于 Windows、macOS、Linux、Android 和 iOS。

内容推荐

Yandex Browser 的一项突出功能是它的内容推荐，它会在您浏览网页时出现。这些建议基于您的浏览历史记录，旨在帮助您发现新的和有趣的网站。

涡轮模式

Yandex 浏览器的另一个有用功能是它的 Turbo 模式，它通过在将网页发送到您的设备之前压缩网页来加快页面加载时间。当您的互联网连接速度较慢或者您只是想更快地加载页面时，这会特别有用。

安全功能

除了性能增强功能外，Yandex 浏览器还具有许多安全功能，可在您在线时保护您。例如，它具有 DNS 欺骗保护功能，有助于防止网站在您不知情的情况下将您重定向到恶意站点。

智能盒子

Yandex 浏览器更独特的功能之一是其 SmartBox，它将搜索栏和地址栏合二为一。这使您可以轻松搜索网站或输入 URL，而无需在两个不同的输入字段之间切换。

客制化

Yandex 浏览器还允许您通过设置自定义背景图像来自定义浏览器的外观。您可以从一系列预定义图像中进行选择，也可以使用您自己的照片。

特点和亮点

Yandex 浏览器是由俄罗斯搜索引擎公司 Yandex 开发的免费网络浏览器。以下是它的一些功能和亮点：

SmartBox： Yandex 浏览器的搜索栏，称为 SmartBox，为搜索查询提供即时建议，并为热门网站提供预测文本输入。

**Turbo 模式：**此功能通过压缩数据和减小网页大小来加快浏览速度。在网速较慢的情况下浏览时特别有用。

DNS 欺骗保护： Yandex 浏览器包括 DNS 欺骗保护以防止网络钓鱼攻击，这是一种窃取用户名、密码和信用卡详细信息等敏感信息的方法。

**Zen：**一个精选的内容提要，显示文章、视频和其他根据用户兴趣定制的内容。

防病毒保护： Yandex 浏览器具有内置的防病毒保护功能，可在用户访问可能有害的网站时发出警告。

**自定义：**浏览器允许用户通过添加扩展、更改浏览器外观和选择不同的搜索引擎来自定义他们的浏览体验。

同步： Yandex 浏览器可以在使用同一 Yandex 帐户登录的设备之间同步数据，例如书签、历史记录和设置。

**语音搜索：**用户可以通过对着麦克风说话而不是打字来搜索网络。

**阅读器模式：**此功能消除了网页的混乱，使阅读文章和其他内容更加容易。

视频的 Turbo 模式： Yandex 浏览器可以自动降低视频质量以节省带宽并减少缓冲时间。
