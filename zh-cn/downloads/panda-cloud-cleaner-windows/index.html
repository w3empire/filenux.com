可疑文件

可疑文件是指已被确定为潜在有害或恶意的文件。这些文件可能包括恶意软件、病毒、特洛伊木马或其他可能危害计算机系统或窃取个人信息的恶意软件类型。可以通过文件扩展名（如 EXE、DLL 或 VBS）或其他特征（如缺少数字签名或存在已知恶意软件模式）来识别可疑文件。

当像 Panda Cloud Cleaner 这样的安全程序或工具扫描计算机时，它会根据已知恶意软件和其他可疑文件的数据库检查系统上的所有文件。如果文件与已知恶意软件匹配或被识别为可疑文件，安全程序将对其进行标记以供进一步分析，并有可能将其从系统中删除。

值得注意的是，某些文件可能被标记为可疑，即使它们不是恶意的。当文件被错误地识别为恶意软件或用户拥有与已知恶意软件相似的合法文件时，就会发生这种情况。在这种情况下，用户可以查看程序的发现并决定是保留还是删除文件。
