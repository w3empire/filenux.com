经常问的问题

什么是 Vivaldi Web 浏览器？

Vivaldi 是一款于 2016 年首次发布的网络浏览器。它是由曾经为另一种流行的网络浏览器 Opera Software 工作的开发人员团队创建的。

Vivaldi 适用于哪些操作系统？

Vivaldi 适用于 Windows、macOS 和 Linux。

是什么让 Vivaldi 与其他网络浏览器不同？

Vivaldi 以其高级定制选项而闻名。它允许用户自定义界面、键盘快捷键、鼠标手势等等。

Vivaldi 是免费的网络浏览器吗？

是的，Vivaldi 是一款免费的网络浏览器。但是，还有一个名为 Vivaldi Pro 的付费版本，其中包括电子邮件和日历集成等附加功能。

Vivaldi 默认使用什么搜索引擎？

Vivaldi 默认使用 Bing 搜索引擎。但是，用户可以将默认搜索引擎更改为他们的选择之一。

Vivaldi 的隐私功能是什么？

Vivaldi 具有多项隐私功能，例如内置广告拦截、跟踪器拦截以及拦截第三方 cookie 的能力。它还具有不保存浏览历史记录或 cookie 的隐私浏览模式。

Vivaldi 可以跨设备同步书签和设置吗？

是的，Vivaldi 有一个同步功能，允许用户在多个设备上同步他们的书签、设置和历史记录。

Vivaldi 有哪些扩展可用？

Vivaldi 支持大多数 Chrome 扩展程序，这些扩展程序可以从 Chrome Web Store 下载。

Vivaldi 的选项卡管理系统是什么样的？

Vivaldi 有一个独特的选项卡管理系统，允许用户在不离开当前选项卡的情况下将选项卡分组、平铺选项卡和预览选项卡。它还具有称为“选项卡堆栈”的功能，允许用户将多个选项卡组合在一个堆栈中。

用户如何获得对 Vivaldi 的支持？

用户可以通过访问 Vivaldi 社区论坛或通过电子邮件联系 Vivaldi 支持来获得对 Vivaldi 的支持。Vivaldi 网站上还有一个知识库和一个常见问题解答部分，可以帮助用户解决常见问题。
