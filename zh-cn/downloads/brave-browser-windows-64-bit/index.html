Brave 是由 Brave Software, Inc. 开发的免费开源网络浏览器。该浏览器于 2016 年首次发布，适用于 Windows、Mac、Linux、iOS 和 Android。

重视隐私和安全的用户会对 Brave 浏览器感到非常满意。该软件程序默认阻止第三方广告和跟踪器，这有助于保护用户隐私并加快页面加载时间。此外，Brave 使用电子前沿基金会 (EFF) 开发的插件 HTTPS Everywhere，自动将所有未加密的 Web 连接升级为安全的 HTTPS 连接。

你说钱？好的，Brave 浏览器也支持加密货币。浏览器有自己的加密货币，称为基本注意令牌（BAT），可用于奖励内容创建者并支持他们的工作。用户可以选择通过观看尊重隐私的广告来赚取 BAT，也可以选择将 BAT 捐赠给他们最喜欢的网站和内容创作者。

除了隐私和加密货币功能外，Brave 还提供许多其他好处。例如，浏览器包含一个内置的密码管理器，可以为用户生成和存储强密码。它还具有称为“屏蔽”的功能，允许用户通过阻止或允许特定类型的内容（例如 cookie、弹出窗口和第三方广告）来自定义他们的浏览体验。

Brave 浏览器的一个独特之处在于它对开源开发的承诺。该浏览器是使用开源工具和技术开发的，公司积极鼓励社区做出贡献。这意味着任何人都可以审查浏览器的开发并为其做出贡献，这有助于确保其安全和透明。

因此，总而言之，Brave 是一款非常重视隐私、安全和加密货币的网络浏览器。它提供了许多有用的功能，例如内置的密码管理器、对 HTTPS 的无处不在的支持以及使用“Shields”自定义浏览行为的能力。此外，作为一个开源项目，Brave 对社区的贡献是透明的和开放的。

Brave 浏览器的历史与演变

在一个在线隐私和安全越来越受追捧的世界里，Brave 浏览器成功地为自己创造了一个独特的利基市场。Brave 于 2016 年推出，现已发展成为Google Chrome、Mozilla Firefox和 Safari 等主流浏览器的流行替代品。凭借对用户隐私的重视、创新的广告拦截技术和用户友好的界面，Brave 赢得了全球数百万用户的青睐。

勇敢浏览器的诞生

Brave 浏览器是 Brendan Eich 的创意，他是一位行业资深人士，也是 JavaScript 编程语言的创造者。在创立 Brave 之前，Eich 在科技界有着漫长而辉煌的职业生涯。他与他人共同创立了 Mozilla，他也是该公司的首席执行官，并在 Firefox 浏览器的开发中发挥了至关重要的作用。

Eich 对 Brave 的愿景是创建一个优先考虑用户隐私和安全的浏览器，同时提供更快的浏览体验。由于对其他浏览器的数据收集做法和侵入性广告感到沮丧，Eich 的目标是构建一个可以保护用户数据并简化在线体验的浏览器。于是，2016 年 1 月，Brave 浏览器诞生了。

Brave 浏览器的演变：关键里程碑和功能

自诞生以来，Brave 浏览器经历了重大的改进和更新。让我们探索其发展过程中的一些关键里程碑和功能：

强调隐私和安全：

从一开始，Brave 的主要关注点就是确保用户隐私和安全。它通过结合内置功能和旨在保护用户免受数据收集和跟踪的默认设置来实现这一点。

Brave 最重要的功能之一是其原生广告拦截器，称为 Brave Shields。Brave Shields 可有效阻止广告和跟踪器，确保用户不会受到侵入性广告的影响，并确保他们的浏览数据保持私密。

除了广告拦截，Brave 还包括 HTTPS Everywhere，这是一项自动将网站连接升级为加密 HTTPS 的功能，有助于保护用户免受潜在的数据拦截和黑客攻击。

更快的浏览体验：

通过阻止广告和跟踪器，Brave 本质上减少了浏览会话期间需要加载的数据量。这带来了更快的浏览体验，页面加载速度明显快于其他浏览器。用户报告说，与主流浏览器相比，速度提升高达 8 倍，尤其是在广告内容繁多的网站上。

Brave Browser 奖励和 Basic Attention Token (BAT) 的引入：

2017 年，Brave 推出了自己的加密货币 Basic Attention Token (BAT)，以帮助重新构想在线广告。BAT 是一种基于以太坊的代币，旨在创建一个更加公平和去中心化的数字广告生态系统。

Brave Rewards 计划允许用户选择观看尊重隐私的广告，以换取获得 BAT 代币。然后，用户可以将他们赚取的 BAT 打赏或贡献给他们最喜欢的内容创作者和网站，甚至可以使用这些代币进行各种在线交易。这种模式通过提供更加透明和注重隐私的广告解决方案来帮助用户和发布商。

Tor 和隐私浏览的集成：

2018 年，Brave 将流行的注重隐私的浏览器 Tor 集成到其隐私浏览模式中。通过 Tor 集成，用户可以在额外的匿名和隐私层下浏览互联网。Tor 网络通过世界各地的多个服务器路由用户流量，使第三方难以追踪用户的在线活动。

移动应用程序开发和跨平台支持：

Brave 迅速将其业务范围扩展到移动平台，推出了适用于 Android 和 iOS 的应用程序。Brave 的移动版本具有与桌面版本相同的隐私和安全功能，为用户提供跨设备的无缝浏览体验。此外，Brave 一直致力于改善移动平台上的用户体验，专注于速度优化和简化的界面设计。

扩展支持和兼容性：

认识到第三方扩展的重要性，Brave 优先支持Chrome 网上应用店中提供的各种扩展和附加组件。这确保了用户可以轻松地从其他浏览器迁移，并且仍然可以使用他们最喜欢的工具和附加组件，而不会损害他们的隐私和安全。

“勇敢的今天”和集成的新闻提要：

为了提高用户参与度，Brave 推出了 Brave Today，这是一种个性化且注重隐私的新闻提要。Brave Today 聚合来自各种来源的内容，同时通过不跟踪阅读习惯或收集数据来维护用户隐私。用户可以根据自己的兴趣和喜好自定义新闻源，提供量身定制的便捷浏览体验。

继续关注隐私和安全功能：

在整个发展过程中，Brave 不断推出新功能和改进，以增强用户隐私和安全性。一些值得注意的新增功能包括防止网站收集唯一设备信息的指纹保护，以及增强的 cookie 阻止选项。

未来之路：Brave Browser 的前景如何？

随着 Brave 浏览器的不断发展，开发团队仍然致力于其隐私、安全和更快浏览体验的基本原则。随着越来越多的用户群和越来越多的内容创作者和发布者开始使用 Basic Attention Token，Brave 有望成为浏览器市场上更加突出的参与者。

此外，随着对数据隐私和安全的担忧越来越普遍，我们可以期待看到 Brave 团队的更多创新。未来的发展可能包括更强大的隐私功能、增强对去中心化网络技术的支持，以及增加基于区块链的解决方案的集成，以增强用户和内容创建者等的能力。

总而言之，Brave 浏览器的旅程是一段鼓舞人心的旅程，它受到更私密、安全和公平的互联网体验愿景的推动。随着浏览器不断发展和获得吸引力，它证明了以用户为中心的设计的力量以及技术重塑数字景观的潜力。

经常问的问题

什么是 Brave Browser 软件程序？

Brave Browser 是一款旨在快速、安全和私密的网络浏览器。它基于 Chromium 浏览器引擎，该引擎与为 Google Chrome 提供支持的引擎相同。

谁创建了 Brave Browser 软件程序？

Brave Browser 由 Brendan Eich 创建，他也是 JavaScript 的创建者和 Mozilla 的联合创始人。

Brave Browser 软件程序与其他浏览器有何不同？

Brave Browser 有一个内置的广告拦截器，可以拦截广告和跟踪器，这可以使浏览速度更快、更私密。它还具有一项名为 Brave Rewards 的功能，允许用户通过观看选择加入的广告来赚取加密货币。

Brave Browser 软件程序可以免费使用吗？

是的，Brave Browser 是免费使用的，可以从 Brave 网站下载。

Brave Browser 软件程序可在哪些平台上使用？

Brave Browser 适用于 Windows、macOS、Linux、Android 和 iOS。

我可以将我的书签从其他浏览器导入到 Brave 网络浏览器吗？

是的，Brave Browser 允许您从其他浏览器（例如 Chrome、Firefox 和 Safari）导入书签。

Brave Browser 软件程序的安全性如何？

Brave Browser 的设计考虑到了安全性，默认情况下使用 HTTPS。它还具有内置的网络钓鱼和恶意软件保护功能，可帮助您确保上网安全。

什么是 Brave Rewards 计划？

Brave Rewards 是一个允许用户通过观看选择加入的广告来赚取加密货币 (BAT) 的程序。然后，用户可以使用赚取的 BAT 来支持他们喜欢的网站或内容创作者。

Brave Browser 软件程序是否支持扩展？

是的，Brave Browser 支持来自 Chrome 网上应用店的扩展程序。

我可以使用 Brave Browser 软件程序跨多个设备同步我的书签和设置吗？

是的，Brave Browser 有一个内置的同步功能，允许您在多个设备上同步您的书签、设置和其他数据。

Brave Browser 的广告拦截器如何工作？

Brave Browser 的广告拦截器使用一种称为“基于区块链的匹配”的技术来拦截广告和跟踪器。这意味着 Brave 不依赖于集中的被阻止域列表，而是使用去中心化系统来识别和阻止广告和跟踪器。

除了广告拦截器之外，Brave Browser 软件程序是否具有任何隐私功能？

是的，除了广告拦截器，Brave Browser 还内置了跟踪器拦截器、指纹保护和脚本拦截器。

我可以自定义 Brave 浏览器的外观吗？

是的，Brave Browser 允许您通过选择不同的主题或创建自己的主题来自定义浏览器的外观。

Brave Browser 是一个开源项目，这意味着任何人都可以为其开发做出贡献。您可以通过报告错误、提交代码或向项目捐款来做出贡献。有关如何贡献的更多信息，请访问 Brave 网站。

什么是网络浏览器？

在当今相互关联的世界中，互联网是我们日常生活中不可或缺的一部分。无论是工作还是休闲，我们都使用 Internet 访问大量信息、娱乐和通信工具。但要做到这一切，我们需要一个网络浏览器——一种让我们访问互联网和查看网页的软件应用程序。

Web 浏览器是一种使我们能够访问网页和其他 Internet 资源的软件。它通过向托管我们要访问的网页的 Web 服务器发送请求来工作。当我们在浏览器的地址栏中输入网址或 URL（统一资源定位器）时，浏览器会向托管该网站的服务器发送请求，服务器通过发回网页数据进行响应，然后浏览器将其显示在我们的显示屏。

Web 浏览器有许多不同的形式，每一种都有自己独特的特性和功能。一些最流行的网络浏览器包括 Google Chrome、Mozilla Firefox、Apple Safari、Microsoft Edge和Opera。这些浏览器中的每一个都有自己独特的界面、特性和功能，但它们都执行相同的基本功能，即允许我们访问和查看网页。

Web 浏览器使用各种技术来显示网页，包括 HTML（超文本标记语言）、CSS（层叠样式表）和 JavaScript。HTML 用于构建网页内容，而 CSS 用于设置内容样式并使其具有视觉吸引力。JavaScript 用于向网页添加交互性，例如动画、下拉菜单和其他动态内容。

现代网络浏览器还配备了一系列附加功能，包括选项卡式浏览、书签、密码管理器和自动填写表单。标签式浏览允许我们在单个浏览器窗口中打开多个网页，从而更容易在它们之间切换。书签让我们保存我们最喜欢的网站以便以后快速访问，而密码管理器存储我们的登录凭据并在我们重新访问这些网站时自动填写它们。

从本质上讲，Web 浏览器是一种必不可少的软件应用程序，它使我们能够访问和查看 Internet 上的网页。在当今的数字时代，它对我们工作、交流和娱乐的能力起着至关重要的作用。凭借其各种特性和功能，Web 浏览器已成为浏览广阔而复杂的 Internet 环境的不可或缺的工具。
