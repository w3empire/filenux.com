Pale Moon Portable 是一款基于 Goanna 布局引擎的免费、开源的 Windows 网络浏览器。它旨在快速、稳定且易于使用，重点是用户友好和可定制。

Pale Moon Portable 旨在便携，允许它从可移动驱动器（如 U 盘）运行，而无需在主机上安装。这使得它非常适合在您不拥有或具有严格安装策略的计算机上使用，以及无论您走到哪里都可以随身携带浏览器。

Pale Moon Portable 的主要优势之一是它使用了 Goanna 布局引擎，该引擎源自 Mozilla 布局引擎。这意味着它能够以类似于 Firefox 和 Chrome 等流行浏览器的方式显示网页，但具有针对速度和效率进行优化的额外优势。

此外，Pale Moon Portable 在构建时考虑到了安全性，包括隐私浏览模式和内置密码管理器等功能，以帮助保护您的在线隐私。

Pale Moon Portable 的另一个独特功能是它支持 NPAPI 插件。NPAPI，即 Netscape 插件应用程序编程接口，是一种遗留的插件架构，过去被许多流行的浏览器插件使用，例如 Java 和 Silverlight。虽然大多数现代浏览器不再支持 NPAPI 插件，但 Pale Moon Portable 仍然提供对它们的支持，对于依赖这些旧插件来开发基于 Web 的应用程序的用户来说，它是一个不错的选择。
