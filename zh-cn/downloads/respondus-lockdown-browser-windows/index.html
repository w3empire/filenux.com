经常问的问题

什么是 Respondus LockDown 浏览器？

Respondus LockDown Browser 是一种自定义浏览器，可锁定学习管理系统 (LMS) 中的测试环境。它旨在通过阻止访问其他应用程序、禁用复制/粘贴和打印以及在在线考试期间捕获音频和视频来防止作弊。

如何下载 Respondus LockDown 浏览器？

Respondus LockDown Browser 通常可通过您机构的学习管理系统或在线门户下载。如果您的机构允许，您也可以直接从 Respondus 网站或从 FileNUX 官方网站下载。

Respondus LockDown 浏览器如何防止作弊？

Respondus LockDown 浏览器通过阻止访问其他应用程序、禁用复制/粘贴和打印以及在在线考试期间捕获音频和视频来防止作弊。它还限制了学生离开考试页面或访问其他网站的能力。

学生可以在移动设备上使用 Respondus LockDown 浏览器吗？

是的，Respondus LockDown 浏览器可用于 iOS 和 Android 设备。但是，由机构决定是否允许使用移动设备进行在线考试。

如果学生的互联网连接在 Respondus LockDown 浏览器中的考试期间丢失会怎样？

如果学生在 Respondus LockDown 浏览器中的考试期间失去互联网连接，学生将能够继续离线进行考试。一旦互联网连接恢复，学生的回答将自动提交。

是否可以绕过 Respondus LockDown 浏览器？

使用未安装浏览器的不同计算机或设备可以绕过 Respondus LockDown 浏览器。然而，这违反了学术诚信，可能会导致严重后果。

Respondus LockDown 浏览器可以用于非考试目的吗？

不，Respondus LockDown 浏览器专门设计用于在线考试。它不能用于一般浏览或访问其他网站。

Respondus LockDown 浏览器的系统要求是什么？

Respondus LockDown Browser 的系统要求因所使用的操作系统和 LMS 而异。通常，浏览器与 Windows 和 Mac 操作系统兼容。

教师如何使用 Respondus LockDown 浏览器设置考试？

教师可以通过其机构的学习管理系统使用 Respondus LockDown 浏览器设置考试。他们通常需要选择要求 Respondus LockDown Browser 进行考试的选项。

Respondus LockDown Browser 可以用于现场考试吗？

不，Respondus LockDown 浏览器专为在线考试而设计。它不适用于现场考试。
