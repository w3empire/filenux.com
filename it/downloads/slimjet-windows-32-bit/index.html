Domande frequenti

Che cos’è il browser Web Slimjet?

Slimjet è un browser Web gratuito e open source basato sul progetto open source Chromium. È stato sviluppato da FlashPeak Inc. e rilasciato nel 2015.

Quali sono i requisiti di sistema per il browser Web Slimjet?

Slimjet supporta i sistemi operativi Windows, Mac e Linux. I requisiti minimi di sistema per Slimjet sono Windows 7, Mac OS X 10.9 o qualsiasi versione a 64 bit di Linux.

Quali sono le caratteristiche del browser Web Slimjet?

Slimjet ha diverse funzionalità, tra cui un blocco degli annunci, un download manager, l’integrazione di Facebook, il caricamento istantaneo di foto e una barra degli strumenti personalizzabile.

Il browser Web Slimjet è sicuro?

Sì, Slimjet è sicuro. Ha funzionalità integrate come la protezione da phishing e malware, che mantengono gli utenti al sicuro durante la navigazione.

Posso importare i miei segnalibri nel browser Web Slimjet?

Sì, puoi facilmente importare segnalibri da altri browser Web come Chrome, Firefox e Internet Explorer su Slimjet.

Il browser Web Slimjet supporta le estensioni?

Sì, Slimjet supporta le estensioni e gli utenti possono scaricare le estensioni dal Chrome Web Store.

Il browser Web Slimjet è gratuito?

Sì, Slimjet è un browser web gratuito. Tuttavia, è disponibile una versione premium per gli utenti che desiderano funzionalità aggiuntive.

Slimjet Web Browser ha una versione premium chiamata “Slimjet VIP”. È una versione a pagamento che offre funzionalità e servizi aggiuntivi come il blocco degli annunci, l’anti-tracciamento e il servizio VPN. La versione premium include anche supporto tecnico prioritario e aggiornamenti.

Gli utenti possono scegliere di acquistare una licenza una tantum o un abbonamento a Slimjet VIP. Il prezzo può variare a seconda del tipo di licenza e della durata dell’abbonamento. Tuttavia, la versione standard di Slimjet Web Browser è gratuita e gli utenti possono comunque usufruire di una varietà di funzioni utili senza dover pagare nulla.

Come si aggiorna il browser Web Slimjet?

Slimjet si aggiorna automaticamente per impostazione predefinita. Tuttavia, puoi anche controllare manualmente gli aggiornamenti nel menu delle impostazioni.

Posso utilizzare il browser Web Slimjet su dispositivi mobili?

No, Slimjet non è attualmente disponibile per i dispositivi mobili. Supporta solo i sistemi operativi desktop.
