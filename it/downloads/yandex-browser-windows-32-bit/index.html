Yandex Browser è un browser Web gratuito sviluppato dalla società tecnologica russa Yandex. È stato rilasciato per la prima volta nel 2012 ed è disponibile per Windows, macOS, Linux, Android e iOS.

Raccomandazioni sui contenuti

Una delle caratteristiche distintive di Yandex Browser sono i consigli sui contenuti, che vengono visualizzati mentre navighi sul Web. Questi consigli si basano sulla tua cronologia di navigazione e hanno lo scopo di aiutarti a scoprire siti web nuovi e interessanti.

Modalità Turbo

Un’altra caratteristica utile di Yandex Browser è la sua modalità Turbo, che accelera i tempi di caricamento delle pagine comprimendo le pagine web prima che vengano inviate al tuo dispositivo. Ciò può essere particolarmente utile quando si dispone di una connessione Internet lenta o se si desidera semplicemente caricare le pagine più velocemente.

Caratteristiche di sicurezza

Oltre alle sue funzionalità di miglioramento delle prestazioni, Yandex Browser ha anche una serie di funzionalità di sicurezza per proteggerti mentre sei online. Ad esempio, ha la protezione dallo spoofing DNS, che aiuta a impedire ai siti Web di reindirizzarti a siti dannosi a tua insaputa.

SmartBox

Una delle caratteristiche più uniche di Yandex Browser è il suo SmartBox, che combina la barra di ricerca e la barra degli indirizzi in una sola. Ciò consente di cercare facilmente siti Web o inserire URL senza dover passare da un campo di immissione all’altro.

Personalizzazione

Yandex Browser ti consente anche di personalizzare l’aspetto del tuo browser impostando un’immagine di sfondo personalizzata. Puoi scegliere tra una selezione di immagini predefinite o utilizzare le tue foto.
