Consigli sui contenuti: Yandex Browser ha una funzione unica che mostra suggerimenti sui contenuti durante la navigazione in Internet. Questi consigli si basano sulla tua cronologia di navigazione e hanno lo scopo di aiutarti a scoprire siti web nuovi e interessanti.

SmartBox: Yandex Browser ha anche una funzione speciale chiamata SmartBox che unisce la barra di ricerca e la barra degli indirizzi in una sola. Ciò consente di cercare facilmente siti Web o inserire URL senza dover passare da un campo di immissione all’altro.
