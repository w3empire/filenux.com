  3. Si vous y êtes invité par le contrôle de compte d’utilisateur (UAC), cliquez sur “Oui” pour permettre au programme d’installation d’apporter des modifications à votre système.
  4. Le programme d’installation lancera alors l’assistant de configuration de Zemana AntiLogger. Cliquez sur “Suivant” pour continuer.

  7. Choisissez si vous souhaitez créer un raccourci sur le bureau pour Zemana AntiLogger et cliquez sur “Suivant”.
  8. Choisissez si vous souhaitez participer au programme d’expérience client Zemana AntiLogger et cliquez sur “Suivant”.
  9. Vérifiez les paramètres d’installation sur l’écran de résumé et cliquez sur “Installer” pour démarrer le processus d’installation.
  10. Attendez que le processus d’installation soit terminé, ce qui peut prendre quelques minutes.
  11. Une fois l’installation terminée, vous verrez un message indiquant que Zemana AntiLogger a été installé avec succès. Cliquez sur “Terminer” pour quitter l’assistant de configuration.
  12. Zemana AntiLogger se lancera alors automatiquement et vous pourrez commencer à utiliser le logiciel pour protéger votre ordinateur contre les logiciels espions et autres menaces de sécurité.
