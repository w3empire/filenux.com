UNDUH
Aman, Gratis dan Cepat

Microsoft Edge adalah browser web yang dikembangkan oleh Microsoft yang pertama kali dirilis pada tahun 2015. Ini dirancang untuk menjadi alternatif yang modern dan lebih cepat untuk Internet Explorer, yang merupakan browser web default pada versi Windows sebelumnya.

Fitur menonjol dari Microsoft Edge adalah kecepatannya. Itu menggunakan mesin rendering baru yang disebut EdgeHTML yang dioptimalkan untuk standar web modern, yang memungkinkannya memuat halaman lebih cepat dan lebih efisien daripada browser lain. Edge juga memiliki antarmuka yang bersih dan sederhana, dengan fitur seperti mode membaca yang menghilangkan gangguan dan memudahkan untuk fokus pada konten yang Anda baca.

Di samping kinerja dan desainnya, Microsoft Edge juga menyediakan berbagai fitur keamanan dan privasi untuk melindungi pengguna saat mereka menjelajahi web. Ini termasuk opsi untuk memblokir pop-up dan cookie yang tidak diinginkan, serta pengelola kata sandi bawaan yang dapat menghasilkan kata sandi yang kuat dan menyimpannya untuk digunakan nanti.

Fitur penting lainnya dari Microsoft Edge adalah integrasinya dengan produk dan layanan Microsoft lainnya. Misalnya, Anda dapat menggunakan browser untuk mengakses akun OneDrive Anda atau menggunakan alat anotasi untuk menandai dan berbagi halaman web dengan orang lain. Microsoft Edge juga mendukung “ekstensi”, yaitu program kecil yang menambahkan fungsionalitas tambahan ke browser.

Singkatnya, Microsoft Edge adalah browser web yang cepat, aman, dan kaya fitur yang sangat cocok untuk penggunaan pribadi dan profesional. Apakah Anda menjelajahi web untuk bersantai atau mengerjakan dokumen penting, Edge memiliki alat dan kemampuan yang Anda perlukan untuk menyelesaikan pekerjaan.

Fitur dan Sorotan

Antarmuka Pengguna: Microsoft Edge hadir dengan antarmuka ramah pengguna yang bersih, modern, dan mudah dinavigasi. Ini memiliki desain minimalis, sehingga memudahkan pengguna untuk fokus pada konten yang mereka lihat.

Manajemen Tab: Edge menyediakan fitur manajemen tab yang sangat baik. Anda dapat dengan mudah membuka, menutup, dan beralih antar tab menggunakan pintasan keyboard, klik mouse, atau gerakan sentuh. Ini juga memungkinkan Anda untuk menyisihkan tab yang saat ini tidak Anda gunakan tetapi ingin diakses nanti.

Tampilan Baca: Microsoft Edge menawarkan opsi “tampilan membaca” yang menghapus iklan dan gangguan dari halaman web, hanya menyisakan teks dan gambar utama artikel. Fitur ini membuat membaca halaman web lebih mudah dan tidak terlalu mengganggu.

Integrasi Cortana: Cortana adalah asisten virtual Microsoft, dan Edge terintegrasi dengan Cortana untuk memberi pengguna fitur tambahan. Misalnya, Anda dapat meminta Cortana untuk membantu Anda menemukan situs web atau mendapatkan informasi tentang suatu topik.

Ekstensi: Edge mendukung ekstensi, memungkinkan pengguna menambahkan fungsionalitas ekstra ke browser. Ada ekstensi untuk pemblokiran iklan, manajemen kata sandi, media sosial, dan banyak lagi.

Dukungan PDF: Edge dapat melihat dan membuat anotasi file PDF tanpa memerlukan perangkat lunak tambahan apa pun. Fitur ini sangat berguna bagi pengguna yang perlu sering bekerja dengan dokumen PDF.

Keamanan: Microsoft Edge memiliki beberapa fitur keamanan yang membantu melindungi pengguna dari situs web berbahaya dan serangan. Ini memiliki filter phishing bawaan, perlindungan malware, dan teknologi kotak pasir yang mengisolasi browser dari sistem operasi.

Antarmuka pengguna

Antarmuka pengguna Microsoft Edge dirancang agar bersih, modern, dan ramah pengguna, dengan fokus pada kesederhanaan dan fungsionalitas. Elemen utama UI meliputi:

Bilah alamat: Terletak di bagian atas jendela browser, bilah alamat memungkinkan pengguna untuk memasukkan URL atau permintaan pencarian. Edge menggunakan Bing sebagai mesin telusur default, tetapi pengguna dapat mengubahnya ke mesin telusur pilihan mereka di pengaturan.

Tab: Di atas bilah alamat, pengguna dapat mengelola beberapa halaman web terbuka menggunakan antarmuka tab. Setiap tab menampilkan judul halaman dan, bila tersedia, favicon untuk memudahkan identifikasi.

Halaman Tab Baru: Saat membuka tab baru, pengguna akan disambut dengan halaman awal yang dapat disesuaikan yang menampilkan bilah pencarian, situs web yang sering dikunjungi, dan konten yang dipersonalisasi, seperti berita dan pembaruan cuaca.

Tombol navigasi: Terletak di sebelah kiri bilah alamat, pengguna dapat menemukan tombol mundur, maju, dan segarkan standar untuk navigasi yang mudah.

Menu: Mengklik ikon tiga titik (elipsis) di sudut kanan atas jendela membuka menu utama, yang menyediakan akses ke berbagai fitur dan pengaturan browser, seperti bookmark, riwayat, ekstensi, opsi privasi dan keamanan, dan lainnya .

Koleksi: Fitur unik Microsoft Edge, Koleksi memungkinkan pengguna mengumpulkan dan mengatur konten web, seperti tautan, gambar, dan teks, di bilah sisi khusus. Fitur ini dapat diakses dengan mengklik ikon Koleksi di sebelah kanan bilah alamat.

Favorit dan Daftar Bacaan: Pengguna dapat menyimpan situs web favorit mereka untuk akses mudah nanti, dan fungsi Daftar Bacaan memungkinkan mereka menyimpan artikel atau halaman web untuk dibaca nanti atau offline.

Ekstensi: Microsoft Edge mendukung berbagai ekstensi, termasuk ekstensi dari Toko Web Chrome, yang dapat meningkatkan fungsionalitas browser dan menyesuaikan pengalaman pengguna.

Fitur aksesibilitas: Edge menawarkan berbagai fitur aksesibilitas, seperti mode pembaca imersif, yang menyederhanakan tata letak halaman web dan menawarkan kemampuan text-to-speech, dan tema kontras tinggi untuk membantu pengguna tunanetra.

Antarmuka pengguna Microsoft Edge dirancang untuk menawarkan pengalaman penelusuran yang mulus dan efisien yang mudah dinavigasi untuk pengguna dari semua tingkat pengalaman.

Manajemen Tab

Microsoft Edge, browser web yang dikembangkan oleh Microsoft, telah memperkenalkan beberapa fitur yang ditujukan untuk meningkatkan pengalaman pengguna, salah satunya adalah manajemen tab. Manajemen tab di Microsoft Edge dirancang untuk membantu pengguna mengatur, menavigasi, dan menangani banyak tab terbuka secara efektif.

Beberapa aspek utama dari fitur manajemen tab Microsoft Edge meliputi:

Tab vertikal: Fitur ini memungkinkan pengguna untuk memindahkan tab terbuka mereka dari tata letak horizontal tradisional ke tumpukan vertikal di sisi jendela. Ini memberikan ikhtisar yang lebih baik tentang tab terbuka, membuatnya lebih mudah untuk menemukan dan beralih di antara tab tersebut. Untuk mengaktifkan tab vertikal, pengguna cukup mengklik tombol “Tab Vertikal” di pojok kiri atas jendela browser.

Grup tab: Grup tab memungkinkan pengguna untuk mengkategorikan dan melabeli tab serupa di bawah satu grup. Ini membantu menjaga browser tetap teratur dan memudahkan navigasi di antara tab terkait. Untuk membuat grup tab, pengguna dapat mengklik kanan tab dan memilih “Tambahkan tab ke grup baru” atau “Tambahkan tab ke grup yang ada”.

Pratinjau tab: Fitur ini menyediakan pratinjau kecil dari konten tab saat mengarahkan mouse ke atasnya. Ini dapat membantu pengguna mengidentifikasi konten tab dengan cepat tanpa perlu beralih ke tab tersebut.

Tab tidur: Microsoft Edge telah memperkenalkan fitur yang disebut “Tab Tidur” untuk membantu menghemat sumber daya sistem. Ketika sebuah tab tidak digunakan untuk jangka waktu tertentu, secara otomatis akan masuk ke mode tidur, membebaskan sumber daya sistem. Pengguna dapat menyesuaikan jangka waktu sebelum tab tidur, dan tab tidur akan dimuat ulang secara otomatis saat diklik.

Menyematkan tab: Pengguna dapat menyematkan situs web yang sering dikunjungi atau tab penting, membuatnya terlihat secara permanen di awal baris tab. Ini membuatnya lebih mudah untuk mengakses tab yang paling penting atau yang sering digunakan.

Pintasan keyboard: Microsoft Edge mendukung berbagai pintasan keyboard untuk manajemen tab, seperti Ctrl+T untuk membuka tab baru, Ctrl+W untuk menutup tab saat ini, dan Ctrl+Tab untuk beralih di antara tab yang terbuka.

Fitur manajemen tab di Microsoft Edge ini secara signifikan meningkatkan pengalaman penelusuran secara keseluruhan, menjadikannya lebih efisien dan teratur bagi pengguna yang mengelola banyak tab sekaligus.

Tampilan Baca

Fitur Tampilan Baca Microsoft Edge adalah alat praktis yang dirancang untuk meningkatkan pengalaman membaca online bagi pengguna. Ini menyederhanakan tata letak halaman web, menghilangkan gangguan seperti iklan, pop-up, dan elemen tidak penting lainnya, membuatnya lebih mudah untuk fokus pada konten utama halaman web.

Berikut adalah beberapa aspek utama dari fitur Tampilan Baca di Microsoft Edge:

Tata letak yang lebih bersih: Dengan menghapus iklan, sidebar, dan kekacauan lainnya, Tampilan Baca menghadirkan antarmuka yang bersih dan tidak berantakan yang membantu pengguna berkonsentrasi pada teks yang mereka baca. Ini bisa sangat membantu saat membaca artikel panjang atau konten padat.

Penyesuaian: Pengguna dapat menyesuaikan tampilan Tampilan Bacaan dengan memilih dari berbagai gaya font, ukuran teks, dan warna latar belakang. Ini memungkinkan pengalaman membaca yang dipersonalisasi yang memenuhi preferensi individu dan kebutuhan aksesibilitas.

Text-to-Speech: Fitur ini menawarkan fungsi text-to-speech bawaan yang disebut “Read Aloud” yang membacakan konten dengan lantang. Ini dapat bermanfaat bagi pengguna tunanetra atau bagi mereka yang lebih suka mendengarkan konten daripada membacanya.

Membaca offline: Tampilan Baca memungkinkan pengguna menyimpan halaman web untuk dibaca secara offline, membuatnya nyaman untuk mengakses konten bahkan saat koneksi internet tidak tersedia.

Akses mudah: Fitur Tampilan Baca dapat dengan mudah diaktifkan dan dinonaktifkan menggunakan ikon buku yang terletak di bilah alamat atau dengan menekan pintasan keyboard Ctrl+Shift+R. Hal ini memudahkan untuk beralih antara tampilan standar dan tampilan yang disederhanakan sesuai kebutuhan.

Tampilan Baca Microsoft Edge adalah alat yang berharga untuk meningkatkan pengalaman pengguna saat membaca konten online. Dengan menawarkan opsi penyesuaian dan fitur aksesibilitas, ini melayani berbagai kebutuhan dan preferensi pengguna.

Integrasi Cortana

Microsoft Edge telah mengintegrasikan asisten virtual perusahaan, Cortana, untuk meningkatkan pengalaman pengguna dan memberikan pengalaman menjelajah yang lebih mulus. Integrasi ini, yang dikenal sebagai fitur Integrasi Cortana, memungkinkan pengguna untuk berinteraksi dengan asisten virtual secara langsung di dalam browser Edge. Meskipun Cortana awalnya diperkenalkan dengan Windows 10, sejak itu telah tersedia untuk platform dan perangkat lain juga.

Integrasi Cortana di Microsoft Edge menawarkan beberapa manfaat dan fitur, antara lain:

Pencarian suara: Pengguna dapat melakukan pencarian suara menggunakan Cortana langsung dari bilah alamat. Fitur ini memungkinkan pengguna untuk mencari informasi tanpa harus mengetikkan query secara manual.

Informasi kontekstual: Cortana dapat memberikan informasi kontekstual untuk teks yang dipilih pada halaman web. Dengan menyorot teks dan mengklik kanan, pengguna dapat mengakses bantuan Cortana dan mendapatkan informasi atau definisi yang relevan tanpa meninggalkan halaman saat ini.

Saran yang dipersonalisasi: Cortana dapat menawarkan saran yang dipersonalisasi berdasarkan riwayat penelusuran, preferensi, dan minat pengguna. Ini dapat membantu pengguna menemukan konten baru dan menghemat waktu saat mencari informasi tertentu.

Pengingat dan pemberitahuan: Cortana dapat mengirimkan pengingat dan pemberitahuan langsung melalui browser Microsoft Edge. Pengguna dapat menyetel pengingat atau menerima peringatan penting tanpa perlu membuka aplikasi atau jendela terpisah.

Integrasi dengan layanan Microsoft lainnya: Cortana bekerja mulus dengan layanan Microsoft lainnya seperti Outlook, OneNote, dan To-Do, memungkinkan pengguna mengelola tugas, janji temu, dan catatan langsung dari browser.

Sinkronisasi lintas perangkat: Cortana dapat menyinkronkan informasi antar perangkat, sehingga pengguna dapat mengakses pengingat, catatan, dan data lain dari smartphone, tablet, atau PC mereka.

Penting untuk dicatat bahwa Microsoft telah mengalihkan fokusnya dari Cortana sebagai asisten virtual mandiri untuk mengintegrasikannya ke dalam skenario produktivitas yang lebih spesifik dalam ekosistem Microsoft. Akibatnya, ketersediaan dan fitur asisten virtual dapat bervariasi tergantung pada wilayah dan perangkat.

Namun, fitur integrasi Cortana di Microsoft Edge tetap menawarkan berbagai manfaat dan kemudahan bagi pengguna, meningkatkan pengalaman menjelajah mereka secara keseluruhan.

Dukungan untuk ekstensi

Ekstensi adalah program perangkat lunak kecil yang dapat mengubah dan meningkatkan fungsionalitas peramban, memungkinkan pengguna menyesuaikan pengalaman menjelajah dengan kebutuhan dan preferensi mereka.

Dukungan Microsoft Edge untuk ekstensi menawarkan beberapa manfaat:

Beragam ekstensi: Karena Microsoft Edge didasarkan pada Chromium, ini dapat mendukung sebagian besar ekstensi Chrome yang tersedia di Toko Web Chrome. Hal ini memudahkan pengguna untuk menemukan dan menginstal ekstensi yang sesuai dengan kebutuhan mereka, karena sudah ada perpustakaan opsi yang besar dan mapan untuk dipilih.

Instalasi mudah: Memasang ekstensi di Microsoft Edge cukup sederhana. Pengguna dapat mengunjungi situs web Pengaya Microsoft Edge atau Toko Web Chrome untuk menjelajahi dan memasang ekstensi. Setelah ekstensi yang diinginkan ditemukan, pengguna dapat mengklik tombol “Tambahkan ke Chrome” atau “Dapatkan” untuk menginstalnya, dan ekstensi tersebut akan ditambahkan ke browser mereka.

Manajemen ekstensi: Microsoft Edge menyediakan antarmuka yang mudah digunakan untuk mengelola ekstensi yang diinstal. Pengguna dapat mengakses halaman Ekstensi dengan mengeklik ikon menu tiga titik di pojok kanan atas browser dan memilih “Ekstensi”. Dari sana, pengguna dapat mengaktifkan, menonaktifkan, atau menghapus ekstensi, serta mengakses setelan untuk masing-masing ekstensi.

Keamanan dan privasi: Microsoft berkomitmen untuk memastikan keamanan dan privasi penggunanya, dan ini mencakup dukungannya untuk ekstensi. Microsoft Edge memiliki proses peninjauan untuk ekstensi yang dikirimkan ke situs web Pengaya, memastikan bahwa mereka memenuhi standar tertentu sebelum tersedia untuk pengguna. Selain itu, pengguna dapat mengontrol izin yang diberikan ke setiap ekstensi, membatasi data yang dapat mereka akses.

Dukungan pengembang: Microsoft menyediakan sumber daya dan dokumentasi untuk pengembang yang tertarik membuat ekstensi untuk Microsoft Edge. Pengembang dapat menggunakan Toolkit Ekstensi Microsoft Edge untuk mengonversi ekstensi Chrome yang ada atau membuat yang baru dari awal. Dukungan ini mendorong pertumbuhan ekosistem ekstensi dan memastikan bahwa pengguna memiliki akses ke berbagai alat dan fitur.

Dukungan PDF

Fungsionalitas PDF Microsoft Edge memberi pengguna cara yang nyaman untuk melihat, menavigasi, membuat anotasi, dan mengelola file PDF tanpa memerlukan perangkat lunak pihak ketiga. Berikut adalah beberapa fitur utama yang ditawarkan Microsoft Edge untuk bekerja dengan dokumen PDF:

Melihat PDF: Microsoft Edge memungkinkan pengguna untuk membuka dan melihat file PDF langsung di dalam browser. File dapat diakses dari drive lokal atau melalui URL.

Navigasi: Browser menawarkan alat navigasi intuitif seperti sidebar untuk melihat thumbnail dokumen, daftar isi, bookmark, dan kemampuan untuk melompat ke halaman tertentu dengan memasukkan nomor halaman.

Opsi Zoom dan Tampilan: Edge menyediakan berbagai tingkat zoom dan opsi tampilan, seperti satu halaman, gulir terus menerus, dan tampilan dua halaman, untuk meningkatkan keterbacaan dan mengakomodasi preferensi pengguna.

Anotasi: Pengguna dapat menambahkan komentar, sorotan, garis bawah, dan coretan ke teks PDF, sehingga memudahkan untuk berkolaborasi atau membuat catatan saat membaca dokumen. Selain itu, pengguna dapat menggambar atau menambahkan bentuk dan kotak teks ke PDF.

Pemilihan dan penyalinan teks: Microsoft Edge memungkinkan pengguna untuk memilih dan menyalin teks dari file PDF, yang kemudian dapat ditempelkan ke aplikasi lain.

Cari: Browser menyertakan fungsi pencarian bawaan yang memungkinkan pengguna menemukan teks atau frasa tertentu dalam dokumen PDF.

Isi dan Tandatangani Formulir: Edge mendukung pengisian dan penandatanganan formulir PDF, memungkinkan pengguna untuk melengkapi dan mengirimkan berbagai jenis formulir langsung di dalam browser.

Aksesibilitas: Microsoft Edge menawarkan dukungan bawaan untuk teknologi bantuan seperti pembaca layar, dan menyediakan opsi untuk mengaktifkan mode kontras tinggi atau membaca keras untuk meningkatkan aksesibilitas bagi pengguna tunanetra.

Simpan dan Cetak: Pengguna dapat dengan mudah menyimpan file PDF ke drive lokal mereka atau mencetaknya langsung dari browser.

Integrasi dengan layanan Microsoft lainnya: Edge terintegrasi secara mulus dengan layanan Microsoft lainnya seperti OneDrive, SharePoint, dan Outlook, memudahkan untuk berbagi, mengakses, dan mengelola file PDF di berbagai platform.

Meskipun Microsoft Edge menyediakan serangkaian fitur yang kuat untuk bekerja dengan dokumen PDF, itu mungkin tidak menyertakan semua fungsi lanjutan yang ditawarkan oleh perangkat lunak PDF khusus atau ekstensi browser pihak ketiga. Tetapi bagi sebagian besar pengguna, dukungan PDF bawaan Edge seharusnya lebih dari cukup untuk tugas sehari-hari dan manajemen dokumen.

Fitur keamanan

Microsoft Edge sangat menekankan keamanan untuk memastikan pengalaman menjelajah yang aman bagi penggunanya. Beberapa fitur keamanan utama yang disertakan dalam Microsoft Edge adalah:

Filter SmartScreen: SmartScreen adalah fitur yang dikembangkan oleh Microsoft yang membantu melindungi pengguna dari serangan phishing, malware, dan perangkat lunak berbahaya lainnya. Itu memeriksa URL situs web dan file terhadap daftar situs berbahaya yang dilaporkan dan unduhan yang terus diperbarui, memperingatkan pengguna jika mereka akan mengunjungi situs web yang berpotensi berbahaya atau mengunduh file yang berpotensi berbahaya.

Sandboxing: Edge menggunakan arsitektur multi-proses yang mengisolasi konten web ke dalam proses kotak pasir. Artinya, jika situs web jahat berhasil mengeksploitasi kerentanan di browser, kerusakannya terbatas, dan penyerang dicegah untuk mengakses data pribadi pengguna atau sumber daya sistem lainnya.

Integrasi Microsoft Defender: Microsoft Edge memiliki integrasi bawaan dengan Microsoft Defender, solusi antivirus dan anti-malware perusahaan. Integrasi ini menyediakan pemindaian unduhan dan file secara real-time yang diakses melalui browser untuk membantu mendeteksi dan memblokir file yang berpotensi berbahaya.

Pencegahan pelacakan: Microsoft Edge menyediakan opsi pencegahan pelacakan untuk memberi pengguna lebih banyak kontrol atas privasi online mereka. Pengguna dapat memilih dari tiga tingkat pencegahan pelacakan yang berbeda: dasar, seimbang, atau ketat. Setiap level memblokir kategori pelacak yang berbeda, dengan pengaturan ketat memberikan tingkat perlindungan privasi tertinggi.

Penjelajahan InPrivate: Mirip dengan mode penjelajahan pribadi browser lain, Penjelajahan InPrivate Edge memungkinkan pengguna untuk menjelajahi web tanpa riwayat penelusuran, file internet sementara, atau cookie disimpan di perangkat mereka.

Pembaruan aman: Microsoft Edge diperbarui secara berkala dengan tambalan dan peningkatan keamanan. Pembaruan ini dikirimkan dengan aman melalui Pembaruan Windows untuk membantu memastikan bahwa pengguna terlindung dari ancaman terbaru.

Pemantauan Kata Sandi: Fitur ini membantu pengguna menjaga keamanan akun online mereka dengan memberi tahu mereka jika kata sandi yang disimpan terlibat dalam pelanggaran data yang diketahui. Pengguna kemudian diminta untuk mengubah kata sandi yang terpengaruh untuk menjaga keamanan akun.

HTTPS secara default: Microsoft Edge secara otomatis memutakhirkan koneksi ke situs web dari HTTP ke HTTPS bila tersedia, memberikan koneksi yang lebih aman dan terenkripsi untuk melindungi data pengguna.

Pertanyaan yang Sering Diajukan

Apa itu browser web Microsoft Edge?

Microsoft Edge adalah browser web yang dikembangkan oleh Microsoft yang tersedia di Windows, macOS, Android, dan iOS.

Kapan Microsoft Edge pertama kali dirilis?

Microsoft Edge pertama kali dirilis pada 29 Juli 2015, sebagai bagian dari Windows 10.

Apa pendahulu dari Microsoft Edge?

Pendahulu Microsoft Edge adalah Internet Explorer.

Apa saja fitur Microsoft Edge?

Beberapa fitur Microsoft Edge termasuk halaman awal yang dapat disesuaikan, ekstensi, mode baca, pembaca PDF bawaan, dan integrasi dengan Cortana (asisten virtual Microsoft).

Bisakah Microsoft Edge digunakan di perangkat seluler?

Ya, Microsoft Edge dapat digunakan di perangkat seluler, termasuk Android dan iOS.

Apa mesin pencari default di Microsoft Edge?

Mesin pencari default di Microsoft Edge adalah Bing, tetapi pengguna dapat mengubahnya ke mesin pencari lain jika mereka mau.

Apa itu fitur Daftar Bacaan di Microsoft Edge?

Fitur Daftar Bacaan di Microsoft Edge memungkinkan pengguna menyimpan halaman web untuk dibaca nanti.

Apa itu “Hub” di Microsoft Edge?

Hub di Microsoft Edge adalah sidebar yang menyediakan akses ke bookmark, daftar bacaan, riwayat, dan unduhan.

Apakah Microsoft Edge mendukung ekstensi?

Ya, Microsoft Edge mendukung ekstensi, yang dapat diunduh dari Microsoft Store.

Bisakah Microsoft Edge menyinkronkan data antar perangkat?

Ya, Microsoft Edge dapat menyinkronkan data antar perangkat, termasuk bookmark, riwayat, kata sandi, dan data formulir.

Apa itu fitur Koleksi di Microsoft Edge?

Fitur Koleksi di Microsoft Edge memungkinkan pengguna untuk menyimpan dan mengatur konten web ke dalam grup.

Apakah Microsoft Edge kompatibel dengan standar web?

Ya, Microsoft Edge kompatibel dengan standar web, termasuk HTML5, CSS3, dan JavaScript.

Apa itu Microsoft Edge DevTools?

Microsoft Edge DevTools adalah seperangkat alat yang dapat digunakan pengembang untuk men-debug dan menguji halaman web.

Bisakah Microsoft Edge memblokir iklan?

Ya, Microsoft Edge dapat memblokir iklan menggunakan fitur pencegahan pelacakan bawaan.

Apakah Microsoft Edge gratis untuk digunakan?

Ya, Microsoft Edge gratis untuk digunakan dan telah diinstal sebelumnya di perangkat Windows.

Informasi lebih lanjut

Microsoft Edge didasarkan pada Chromium, sebuah proyek peramban sumber terbuka yang juga menjadi dasar Google Chrome. Pada bulan Desember 2018, Microsoft mengumumkan akan membangun kembali Edge menggunakan mesin Chromium untuk meningkatkan kompatibilitas dan kinerja. Versi baru Edge dirilis pada Januari 2020, dan sekarang didasarkan pada teknologi inti yang sama dengan Google Chrome.

Meskipun Microsoft Edge dan Google Chrome memiliki fondasi yang sama di Chromium, mereka masih merupakan browser yang berbeda dengan fitur dan desain uniknya sendiri. Misalnya, Edge menyertakan mode membaca dan alat anotasi yang tidak ditemukan di Chrome, dan juga terintegrasi dengan produk dan layanan Microsoft lainnya. Namun, kedua browser tersebut dapat menjalankan ekstensi dan aplikasi web yang sama, dan keduanya juga memiliki fitur kinerja dan keamanan yang serupa.
