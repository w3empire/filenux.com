Discover What's New in Firefox 113: Search and Picture-in-Picture Improvements and Tighter Security Measures!
Explore Firefox 113's updated features: Improved search, enhanced PiP, stronger security, revamped extensions menu, performance upgrades, and customizable toolbar.

By Marc Alfano
Updated: May 13, 2023

Mozilla has released the latest version of its popular web browser, Firefox 113, with significant updates in security, search functionality, and picture-in-picture mode.

Firefox has long been a popular choice for web browsing enthusiasts with its open-source ethos and privacy-focused approach. With version 113, Mozilla aims to further enhance the user experience by offering several new features.

One of the key updates in Firefox 113 is the search functionality. Mozilla has integrated new search suggestions and results powered by Microsoft’s Bing search engine. This integration makes it easier for users to find relevant information without having to switch to a separate search engine. Firefox 113 also includes a new search widget, which appears on the new tab page, making it easier to search the web directly from the browser’s home screen.

Another notable feature of Firefox 113 is the picture-in-picture mode. This mode allows users to watch videos in a separate window while they continue to browse other websites. This feature was first introduced in Firefox 71 but has been improved in the latest version. Users can now resize the picture-in-picture window and control the playback speed of the video.

In addition to these new features, Firefox 113 also comes with tighter security measures. Mozilla has enhanced the browser’s tracking protection feature, making it even more difficult for websites to track users’ online activities. The browser now includes a new privacy dashboard that displays a summary of the trackers that have been blocked in the current session. Users can also access more detailed information about tracking protection by clicking on the shield icon in the address bar.

Firefox 113 also includes several other security-related improvements. The browser now supports the latest version of the Transport Layer Security (TLS) protocol, which provides a more secure connection between the browser and websites. Mozilla has also improved the browser’s anti-fingerprinting capabilities, making it more difficult for websites to identify and track users based on their browser and device configurations.

So, to sum up, Firefox 113 is an impressive update that brings several new features and improvements to the table. The integration of Microsoft Bing search and the new picture-in-picture mode are welcome additions that enhance the browsing experience. The tighter security measures are also a significant improvement, providing users with more privacy and protection online. If you’re a Firefox user, the latest version is definitely worth a download.

More improvements

Alongside the search and picture-in-picture improvements and tighter security measures, Firefox 113 also comes with several other enhancements that improve the overall user experience.

One of the most notable improvements is the redesigned extensions menu. The new menu provides users with easier access to their installed extensions and allows them to quickly enable or disable them. The menu also displays information about each extension, including its name, author, and a brief description.

Firefox 113 also includes a new bookmark toolbar menu that makes it easier to organize and access bookmarks. Users can now right-click on the bookmark toolbar to open the new menu, which displays all of their bookmarks in a dropdown list. This menu also allows users to create new bookmarks, folders, and separators directly from the toolbar.

Another new feature in Firefox 113 is the ability to customize the toolbar. Users can now choose which icons they want to display on the toolbar and in what order. This customization option allows users to tailor the toolbar to their specific needs and preferences.

Firefox 113 also includes several performance improvements. The browser now loads web pages faster, thanks to a new cache management system. The new system improves the performance of frequently visited websites by caching them more efficiently, reducing the time it takes to load them.

Besides these improvements, Firefox 113 also comes with several accessibility enhancements. The browser now supports the Accessible Name and Description Computation (ANDC) specification, which improves the accessibility of web pages for users with disabilities. The browser also includes new options for adjusting the font size and contrast, making it easier for users to read web pages.

Firefox 113 is a significant update that brings several new features and improvements to the browser. The redesigned extensions and bookmark toolbar menus, toolbar customization options, performance improvements, and accessibility enhancements all contribute to a more user-friendly and efficient browsing experience. If you’re a Firefox user, you’ll definitely want to upgrade to the latest version.
