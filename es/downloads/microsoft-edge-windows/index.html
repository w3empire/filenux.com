Seguro, Gratis y Rápido

Microsoft Edge es un navegador web desarrollado por Microsoft que se lanzó por primera vez en 2015. Está diseñado para ser una alternativa moderna y más rápida a Internet Explorer, que era el navegador web predeterminado en versiones anteriores de Windows.

Una característica destacada de Microsoft Edge es su velocidad. Utiliza un nuevo motor de renderizado llamado EdgeHTML que está optimizado para los estándares web modernos, lo que le permite cargar páginas de manera más rápida y eficiente que otros navegadores. Edge también tiene una interfaz limpia y simple, con características como un modo de lectura que elimina las distracciones y facilita concentrarse en el contenido que está leyendo.

Además de su rendimiento y diseño, Microsoft Edge también ofrece una gama de características de seguridad y privacidad para proteger a los usuarios mientras navegan por la web. Estos incluyen opciones para bloquear ventanas emergentes y cookies no deseadas, así como un administrador de contraseñas integrado que puede generar contraseñas seguras y guardarlas para su uso posterior.

Otra característica notable de Microsoft Edge es su integración con otros productos y servicios de Microsoft. Por ejemplo, puede usar el navegador para acceder a su cuenta de OneDrive o usar las herramientas de anotación para marcar y compartir páginas web con otras personas. Microsoft Edge también admite “extensiones”, que son pequeños programas que agregan funcionalidad adicional al navegador.

En resumen, Microsoft Edge es un navegador web rápido, seguro y rico en funciones que es adecuado tanto para uso personal como profesional. Ya sea que esté navegando por la web por placer o trabajando en documentos importantes, Edge tiene las herramientas y capacidades que necesita para hacer el trabajo.

Características y aspectos destacados

Interfaz de usuario: Microsoft Edge viene con una interfaz fácil de usar que es limpia, moderna y fácil de navegar. Tiene un diseño minimalista, lo que facilita que los usuarios se concentren en el contenido que están viendo.

Gestión de pestañas: Edge ofrece excelentes funciones de gestión de pestañas. Puede abrir, cerrar y cambiar fácilmente entre pestañas mediante métodos abreviados de teclado, clics del mouse o gestos táctiles. También le permite dejar de lado las pestañas que no está usando actualmente pero a las que desea acceder más tarde.

Vista de lectura: Microsoft Edge ofrece una opción de “vista de lectura” que elimina anuncios y distracciones de una página web, dejando solo el texto principal y las imágenes del artículo. Esta característica hace que la lectura de páginas web sea más fácil y menos distraída.

Integración de Cortana: Cortana es el asistente virtual de Microsoft y Edge se integra con Cortana para brindar a los usuarios funciones adicionales. Por ejemplo, puede pedirle a Cortana que lo ayude a encontrar un sitio web u obtener información sobre un tema.

Extensiones: Edge admite extensiones, lo que permite a los usuarios agregar funciones adicionales al navegador. Hay extensiones para bloqueo de anuncios, administración de contraseñas, redes sociales y mucho más.

Compatibilidad con PDF: Edge puede ver y anotar archivos PDF sin necesidad de ningún software adicional. Esta función es particularmente útil para los usuarios que necesitan trabajar con documentos PDF con frecuencia.

Seguridad: Microsoft Edge tiene varias características de seguridad que ayudan a proteger a los usuarios de sitios web y ataques maliciosos. Tiene un filtro de phishing incorporado, protección contra malware y tecnología de sandboxing que aísla el navegador del sistema operativo.

Interfaz de usuario

La interfaz de usuario de Microsoft Edge está diseñada para ser limpia, moderna y fácil de usar, con un enfoque en la simplicidad y la funcionalidad. Los elementos clave de la interfaz de usuario incluyen:

Barra de direcciones: Ubicada en la parte superior de la ventana del navegador, la barra de direcciones permite a los usuarios ingresar URL o consultas de búsqueda. Edge usa Bing como su motor de búsqueda predeterminado, pero los usuarios pueden cambiarlo a su motor de búsqueda preferido en la configuración.

Pestañas: encima de la barra de direcciones, los usuarios pueden administrar varias páginas web abiertas mediante una interfaz con pestañas. Cada pestaña muestra el título de la página y, cuando está disponible, un favicon para facilitar la identificación.

Página de nueva pestaña: al abrir una nueva pestaña, los usuarios son recibidos con una página de inicio personalizable que muestra una barra de búsqueda, sitios web visitados con frecuencia y contenido personalizado, como noticias y actualizaciones del clima.

Botones de navegación: ubicados a la izquierda de la barra de direcciones, los usuarios pueden encontrar los botones estándar atrás, adelante y actualizar para facilitar la navegación.

Menú: al hacer clic en el ícono de tres puntos (puntos suspensivos) en la esquina superior derecha de la ventana, se abre el menú principal, que brinda acceso a varias funciones y configuraciones del navegador, como marcadores, historial, extensiones, opciones de privacidad y seguridad, y más. .

Colecciones: una característica única de Microsoft Edge, las colecciones permiten a los usuarios recopilar y organizar contenido web, como enlaces, imágenes y texto, en una barra lateral dedicada. Se puede acceder a esta función haciendo clic en el icono Colecciones a la derecha de la barra de direcciones.

Favoritos y lista de lectura: los usuarios pueden guardar sus sitios web favoritos para acceder fácilmente a ellos más tarde, y la función Lista de lectura les permite guardar artículos o páginas web para leerlos más tarde o sin conexión.

Extensiones: Microsoft Edge admite una amplia gama de extensiones, incluidas las de Chrome Web Store, que pueden mejorar la funcionalidad del navegador y personalizar la experiencia del usuario.

Funciones de accesibilidad: Edge ofrece varias funciones de accesibilidad, como el modo de lector inmersivo, que simplifica los diseños de páginas web y ofrece capacidades de texto a voz y temas de alto contraste para ayudar a los usuarios con discapacidades visuales.

La interfaz de usuario de Microsoft Edge está diseñada para ofrecer una experiencia de navegación fluida y eficiente que es fácil de navegar para usuarios de todos los niveles de experiencia.

Gestión de pestañas

Microsoft Edge, el navegador web desarrollado por Microsoft, ha introducido varias características destinadas a mejorar la experiencia del usuario, siendo la gestión de pestañas una de ellas. La administración de pestañas en Microsoft Edge está diseñada para ayudar a los usuarios a organizar, navegar y manejar múltiples pestañas abiertas de manera efectiva.

Algunos aspectos clave de la función de administración de pestañas de Microsoft Edge incluyen:

Pestañas verticales: esta característica permite a los usuarios mover sus pestañas abiertas del diseño horizontal tradicional a una pila vertical en el costado de la ventana. Esto proporciona una mejor visión general de las pestañas abiertas, lo que facilita encontrarlas y cambiar entre ellas. Para habilitar las pestañas verticales, los usuarios simplemente pueden hacer clic en el botón “Pestañas verticales” en la esquina superior izquierda de la ventana del navegador.

Grupos de pestañas: los grupos de pestañas permiten a los usuarios categorizar y etiquetar pestañas similares en un solo grupo. Esto ayuda a mantener el navegador organizado y facilita la navegación entre pestañas relacionadas. Para crear un grupo de pestañas, los usuarios pueden hacer clic derecho en una pestaña y seleccionar “Agregar pestaña a un nuevo grupo” o “Agregar pestaña a un grupo existente”.

Vista previa de la pestaña: esta función proporciona una pequeña vista previa del contenido de una pestaña al pasar el mouse sobre ella. Esto puede ayudar a los usuarios a identificar rápidamente el contenido de una pestaña sin necesidad de cambiar a ella.

Pestañas para dormir: Microsoft Edge ha introducido una característica llamada “Pestañas para dormir” para ayudar a conservar los recursos del sistema. Cuando una pestaña no está en uso durante un cierto período de tiempo, automáticamente entrará en modo de suspensión, liberando recursos del sistema. Los usuarios pueden personalizar el período de tiempo antes de que una pestaña se duerma, y ​​las pestañas dormidas se recargarán automáticamente cuando se haga clic en ellas.

Anclar pestañas: los usuarios pueden anclar sitios web visitados con frecuencia o pestañas importantes, haciéndolos visibles permanentemente al comienzo de la fila de pestañas. Esto facilita el acceso a las pestañas más importantes o de uso frecuente.

Métodos abreviados de teclado: Microsoft Edge admite varios métodos abreviados de teclado para la administración de pestañas, como Ctrl+T para abrir una nueva pestaña, Ctrl+W para cerrar la pestaña actual y Ctrl+Tab para cambiar entre pestañas abiertas.

Estas características de administración de pestañas en Microsoft Edge mejoran significativamente la experiencia general de navegación, haciéndola más eficiente y organizada para los usuarios que administran varias pestañas a la vez.

Vista de lectura

La función Vista de lectura de Microsoft Edge es una herramienta útil diseñada para mejorar la experiencia de lectura en línea de los usuarios. Simplifica el diseño de las páginas web, eliminando distracciones como anuncios, ventanas emergentes y otros elementos no esenciales, lo que facilita concentrarse en el contenido principal de una página web.

Estos son algunos aspectos clave de la función Vista de lectura en Microsoft Edge:

Diseño más limpio: al eliminar anuncios, barras laterales y otros elementos desordenados, la Vista de lectura presenta una interfaz limpia y despejada que ayuda a los usuarios a concentrarse en el texto que están leyendo. Esto puede ser particularmente útil al leer artículos largos o contenido denso.

Personalización: los usuarios pueden personalizar la apariencia de la Vista de lectura eligiendo entre diferentes estilos de fuente, tamaños de texto y colores de fondo. Esto permite una experiencia de lectura personalizada que se adapta a las preferencias individuales y las necesidades de accesibilidad.

Texto a voz: la función ofrece una función integrada de texto a voz llamada “Leer en voz alta” que lee el contenido en voz alta. Esto puede ser útil para usuarios con problemas de visión o para aquellos que prefieren escuchar el contenido en lugar de leerlo.

Lectura sin conexión: la vista de lectura permite a los usuarios guardar páginas web para leerlas sin conexión, lo que facilita el acceso al contenido incluso cuando no hay una conexión a Internet disponible.

Fácil acceso: la función Vista de lectura se puede activar y desactivar fácilmente usando el icono de libro ubicado en la barra de direcciones o presionando el atajo de teclado Ctrl+Shift+R. Esto simplifica el cambio entre las vistas estándar y simplificada según sea necesario.

La vista de lectura de Microsoft Edge es una herramienta valiosa para mejorar la experiencia del usuario al leer contenido en línea. Al ofrecer opciones de personalización y funciones de accesibilidad, satisface una amplia gama de necesidades y preferencias de los usuarios.

Integración de Cortana

Microsoft Edge ha integrado el asistente virtual de la empresa, Cortana, para mejorar la experiencia del usuario y brindar una experiencia de navegación más fluida. Esta integración, conocida como la función Integración de Cortana, permite a los usuarios interactuar con el asistente virtual directamente dentro del navegador Edge. Aunque Cortana se introdujo inicialmente con Windows 10, desde entonces también está disponible para otras plataformas y dispositivos.

La integración de Cortana en Microsoft Edge ofrece varios beneficios y características, que incluyen:

Búsqueda por voz: los usuarios pueden realizar búsquedas por voz usando Cortana directamente desde la barra de direcciones. Esta función permite a los usuarios buscar información sin tener que escribir consultas manualmente.

Información contextual: Cortana puede proporcionar información contextual para el texto seleccionado en una página web. Al resaltar el texto y hacer clic con el botón derecho, los usuarios pueden acceder a la asistencia de Cortana y obtener información o definiciones relevantes sin salir de la página actual.

Sugerencias personalizadas: Cortana puede ofrecer sugerencias personalizadas basadas en el historial de navegación, las preferencias y los intereses del usuario. Esto puede ayudar a los usuarios a descubrir nuevos contenidos y ahorrar tiempo al buscar información específica.

Recordatorios y notificaciones: Cortana puede enviar recordatorios y notificaciones directamente a través del navegador Microsoft Edge. Los usuarios pueden configurar recordatorios o recibir alertas importantes sin necesidad de abrir una aplicación o ventana separada.

Integración con otros servicios de Microsoft: Cortana funciona a la perfección con otros servicios de Microsoft como Outlook, OneNote y To-Do, lo que permite a los usuarios administrar tareas, citas y notas directamente desde el navegador.

Sincronización entre dispositivos: Cortana puede sincronizar información entre dispositivos, para que los usuarios puedan acceder a recordatorios, notas y otros datos desde sus teléfonos inteligentes, tabletas o PC.

Es importante tener en cuenta que, desde entonces, Microsoft ha cambiado su enfoque de Cortana como un asistente virtual independiente para integrarlo en escenarios de productividad más específicos dentro del ecosistema de Microsoft. En consecuencia, la disponibilidad y las funciones del asistente virtual pueden variar según la región y el dispositivo.

Sin embargo, la característica de integración de Cortana en Microsoft Edge aún ofrece una gama de beneficios y conveniencias para los usuarios, mejorando su experiencia de navegación general.

Soporte para extensiones

Las extensiones son pequeños programas de software que pueden modificar y mejorar la funcionalidad del navegador, lo que permite a los usuarios adaptar su experiencia de navegación a sus necesidades y preferencias.

El soporte de Microsoft Edge para extensiones ofrece varios beneficios:

Amplia gama de extensiones: dado que Microsoft Edge se basa en Chromium, puede admitir la gran mayoría de las extensiones de Chrome disponibles en Chrome Web Store. Esto facilita que los usuarios encuentren e instalen extensiones que se adapten a sus necesidades, ya que ya existe una biblioteca grande y establecida de opciones para elegir.

Fácil instalación: Instalar extensiones en Microsoft Edge es bastante sencillo. Los usuarios pueden visitar el sitio web de complementos de Microsoft Edge o Chrome Web Store para buscar e instalar extensiones. Una vez que se encuentra la extensión deseada, los usuarios pueden hacer clic en el botón “Agregar a Chrome” u “Obtener” para instalarla y la extensión se agregará a su navegador.

Administración de extensiones: Microsoft Edge proporciona una interfaz fácil de usar para administrar las extensiones instaladas. Los usuarios pueden acceder a la página Extensiones haciendo clic en el icono de menú de tres puntos en la esquina superior derecha del navegador y seleccionando “Extensiones”. Desde allí, los usuarios pueden habilitar, deshabilitar o eliminar extensiones, así como acceder a la configuración de extensiones individuales.

Seguridad y privacidad: Microsoft se compromete a garantizar la seguridad y privacidad de sus usuarios, y esto se extiende a su soporte para extensiones. Microsoft Edge tiene un proceso de revisión para las extensiones enviadas a su sitio web de complementos, lo que garantiza que cumplan con ciertos estándares antes de que estén disponibles para los usuarios. Además, los usuarios pueden controlar los permisos otorgados a cada extensión, limitando los datos a los que pueden acceder.

Soporte para desarrolladores: Microsoft proporciona recursos y documentación para los desarrolladores interesados ​​en crear extensiones para Microsoft Edge. Los desarrolladores pueden usar el kit de herramientas de extensión de Microsoft Edge para convertir las extensiones de Chrome existentes o crear otras nuevas desde cero. Este soporte fomenta el crecimiento del ecosistema de extensiones y garantiza que los usuarios tengan acceso a una amplia variedad de herramientas y funciones.

Soporte PDF

La funcionalidad de PDF de Microsoft Edge brinda a los usuarios una forma conveniente de ver, navegar, anotar y administrar archivos PDF sin necesidad de software de terceros. Estas son algunas de las características clave que ofrece Microsoft Edge para trabajar con documentos PDF:

Visualización de archivos PDF: Microsoft Edge permite a los usuarios abrir y ver archivos PDF directamente en el navegador. Se puede acceder a los archivos desde una unidad local o mediante una URL.

Navegación: el navegador ofrece herramientas de navegación intuitivas, como una barra lateral para ver miniaturas de documentos, una tabla de contenido, marcadores y la capacidad de saltar a páginas específicas ingresando el número de página.

Opciones de vista y zoom: Edge ofrece una gama de niveles de zoom y opciones de vista, como una sola página, desplazamiento continuo y vista de dos páginas, para mejorar la legibilidad y adaptarse a las preferencias del usuario.

Anotaciones: los usuarios pueden agregar comentarios, resaltar, subrayar y tachar el texto del PDF, lo que facilita la colaboración o la toma de notas mientras lee un documento. Además, los usuarios pueden dibujar o agregar formas y cuadros de texto al PDF.

Selección y copia de texto: Microsoft Edge permite a los usuarios seleccionar y copiar texto de archivos PDF, que luego se pueden pegar en otras aplicaciones.

Búsqueda: el navegador incluye una función de búsqueda integrada que permite a los usuarios encontrar texto o frases específicas dentro del documento PDF.

Rellenar y firmar formularios: Edge admite rellenar y firmar formularios PDF, lo que permite a los usuarios completar y enviar varios tipos de formularios directamente desde el navegador.

Accesibilidad: Microsoft Edge ofrece soporte integrado para tecnologías de asistencia, como lectores de pantalla, y brinda opciones para habilitar el modo de alto contraste o leer en voz alta para mejorar la accesibilidad para usuarios con discapacidades visuales.

Guardar e imprimir: los usuarios pueden guardar fácilmente archivos PDF en su unidad local o imprimirlos directamente desde el navegador.

Integración con otros servicios de Microsoft: Edge se integra a la perfección con otros servicios de Microsoft, como OneDrive, SharePoint y Outlook, lo que facilita compartir, acceder y administrar archivos PDF en varias plataformas.

Si bien Microsoft Edge proporciona un sólido conjunto de características para trabajar con documentos PDF, es posible que no incluya todas las funcionalidades avanzadas que ofrecen el software PDF dedicado o las extensiones de navegador de terceros. Pero para la mayoría de los usuarios, la compatibilidad con PDF integrada de Edge debería ser más que suficiente para las tareas diarias y la gestión de documentos.

Características de seguridad

Microsoft Edge pone un fuerte énfasis en la seguridad para garantizar una experiencia de navegación segura para sus usuarios. Algunas de las funciones de seguridad clave incluidas en Microsoft Edge son:

Filtro SmartScreen: SmartScreen es una característica desarrollada por Microsoft que ayuda a proteger a los usuarios de ataques de phishing, malware y otro software malicioso. Comprueba las direcciones URL de los sitios web y los archivos con una lista constantemente actualizada de descargas y sitios maliciosos informados, alertando a los usuarios si están a punto de visitar un sitio web potencialmente peligroso o descargar un archivo potencialmente dañino.

Sandboxing: Edge utiliza una arquitectura multiproceso que aísla el contenido web en procesos de espacio aislado. Esto significa que si un sitio web malicioso logra explotar una vulnerabilidad en el navegador, el daño es limitado y se evita que el atacante acceda a los datos personales del usuario u otros recursos del sistema.

Integración de Microsoft Defender: Microsoft Edge tiene una integración integrada con Microsoft Defender, la solución antivirus y antimalware de la empresa. Esta integración proporciona análisis en tiempo real de descargas y archivos a los que se accede a través del navegador para ayudar a detectar y bloquear archivos potencialmente dañinos.

Prevención de seguimiento: Microsoft Edge ofrece opciones de prevención de seguimiento para dar a los usuarios más control sobre su privacidad en línea. Los usuarios pueden elegir entre tres niveles diferentes de prevención de seguimiento: básico, equilibrado o estricto. Cada nivel bloquea diferentes categorías de rastreadores, y la configuración estricta proporciona el nivel más alto de protección de la privacidad.

Navegación InPrivate: similar a los modos de navegación privada de otros navegadores, la navegación InPrivate de Edge permite a los usuarios navegar por la web sin que su historial de navegación, archivos temporales de Internet o cookies se almacenen en sus dispositivos.

Actualizaciones seguras: Microsoft Edge se actualiza regularmente con parches de seguridad y mejoras. Estas actualizaciones se entregan de forma segura a través de Windows Update para ayudar a garantizar que los usuarios estén protegidos contra las amenazas más recientes.

Monitor de contraseñas: esta característica ayuda a los usuarios a mantener seguras sus cuentas en línea al alertarlos si sus contraseñas guardadas han estado involucradas en una violación de datos conocida. Luego se le pide al usuario que cambie la contraseña afectada para mantener la seguridad de la cuenta.

HTTPS de forma predeterminada: Microsoft Edge actualiza automáticamente las conexiones a sitios web de HTTP a HTTPS cuando está disponible, lo que proporciona una conexión más segura y cifrada para proteger los datos de los usuarios.

Más información

Microsoft Edge se basa en Chromium, un proyecto de navegador de código abierto que también es la base de Google Chrome. En diciembre de 2018, Microsoft anunció que reconstruiría Edge utilizando el motor Chromium para mejorar la compatibilidad y el rendimiento. La nueva versión de Edge se lanzó en enero de 2020 y ahora se basa en la misma tecnología central que Google Chrome.

Si bien Microsoft Edge y Google Chrome comparten una base común en Chromium, siguen siendo navegadores distintos con sus propias características y diseños únicos. Por ejemplo, Edge incluye un modo de lectura y herramientas de anotación que no se encuentran en Chrome, y también se integra con otros productos y servicios de Microsoft. Sin embargo, los dos navegadores pueden ejecutar las mismas extensiones y aplicaciones web, y también tienen características similares de rendimiento y seguridad.

Firefox es de código abierto, lo que significa que lo creó una comunidad de desarrolladores apasionados por crear un navegador gratuito y accesible para todos. Y seamos realistas, el logo del pequeño zorro es demasiado lindo para resistirse.

Luego está Microsoft Edge, el navegador que nadie pidió pero que obtuvimos de todos modos. Antes se llamaba Internet Explorer, pero Microsoft se dio cuenta de que nadie quería usar un navegador que fuera más lento que un perezoso los lunes por la mañana. Así que lo renombraron y le dieron un lavado de cara. No está mal, pero tampoco es genial. Es como el hijo del medio de los navegadores, siempre tratando de complacer a todos y fallando.

Por último, pero no menos importante, tenemos Safari. Es el navegador para los fanáticos de Apple que simplemente no pueden tener suficiente de esa dulce, dulce bondad de Cupertino. Es rápido, elegante y tiene todas las funciones que necesita para navegar por Internet en su Mac o iPhone. Pero seamos realistas: la única razón por la que la gente usa Safari es porque les da pereza descargar Chrome.

Entonces, ahí lo tienen, amigos, un resumen rápido de los navegadores web más populares. Ya sea que sea un fanático de Chrome o un partidario de Firefox, todos podemos estar de acuerdo en una cosa: estaríamos perdidos sin nuestro navegador de confianza. Después de todo, ¿de qué otra manera podríamos posponer las cosas en el trabajo todo el día?
