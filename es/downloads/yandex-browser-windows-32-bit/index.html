Recomendaciones de contenido: Yandex Browser tiene una característica única que muestra sugerencias de contenido mientras navega por Internet. Estas recomendaciones se basan en su historial de navegación y están destinadas a ayudarlo a descubrir sitios web nuevos e interesantes.

SmartBox: Yandex Browser también tiene una función especial llamada SmartBox que fusiona la barra de búsqueda y la barra de direcciones en una sola. Esto le permite buscar fácilmente sitios web o ingresar URL sin tener que cambiar entre dos campos de entrada diferentes.
