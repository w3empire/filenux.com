ダウンロード
安全、無料、高速

Respondus LockDown Browser は、オンライン試験の受験専用に設計された Web ブラウザです。これは、学校、大学、およびその他の教育機関で、オンライン評価の完全性とセキュリティを確保するために一般的に使用されています。ブラウザは、学生がテスト中に他のリソースやアプリケーションにアクセスできないように設計されているため、カンニングを防ぎ、試験の結果が正確で信頼できるものになります。

Respondus LockDown Browser の主な機能の 1 つは、学生がオンライン試験に参加している間、他の Web サイトまたはプログラムへのアクセスを制限することです。これは、デスクトップ、タスクバー、およびその他のシステム リソースへのアクセスをブロックするなど、さまざまなセキュリティ対策を使用することで実現されます。

学生が Respondus LockDown ブラウザを開くと、試験のログイン ページが自動的に表示されます。ログインすると、試験が完了して提出されるまで、他のリソースやアプリケーションにアクセスできません。

Respondus LockDown Browser のもう 1 つの重要な機能は、学生が試験から他のアプリケーションにテキストをコピー、貼り付け、または転送することを防止することです。

これにより、学生が試験の質問や回答を他の人と共有できないようにすることができます。これにより、評価の完全性が損なわれる可能性があります。さらに、ブラウザは学生が試験資料を印刷または保存できないように設計されているため、不正行為をさらに防止できます。

Respondus LockDown ブラウザを使用することで、オンライン試験のセキュリティと完全性を確保できます。ブラウザは、学生がテストを受けている間、他のリソースやアプリケーションにアクセスできないようにするため、カンニングをしたり、試験資料を他人と共有したりすることははるかに困難です。これにより、試験結果の信頼性と正確性が保証されます。これは、学生と評価を管理する教育者の両方にとって重要です。

Respondus LockDown Browser のもう 1 つの利点は、使いやすいことです。ブラウザは Windows と Mac の両方のプラットフォームで利用でき、わずか数分でダウンロードしてコンピュータにインストールできます。ブラウザをインストールしたら、学生は通常のログイン資格情報を使用して試験にログインするだけです。Respondus LockDown Browser のインターフェイスはわかりやすく、ユーザー フレンドリーであるため、学生は簡単にナビゲートして試験を完了することができます。

結論として、Respondus LockDown Browser は、オンライン試験を受けるために特別に設計された特殊な Web ブラウザーです。テスト中に学生が他のリソースやアプリケーションにアクセスできないようにすることで、オンライン評価のセキュリティと整合性を確保するのに役立ちます。使いやすく、Windows と Mac の両方のプラットフォームで利用できます。Respondus LockDown Browser を使用することで、学校や大学は、オンライン試験の結果が正確で信頼できるものであることに自信を持つことができます。

機能とハイライト

試験環境をロックダウン: Respondus LockDown Browser は、試験中の学生の他のアプリケーションや Web サイトへのアクセスを制限し、カンニングや不正な資料へのアクセスを防ぎます。

**印刷およびコピー機能の防止:**このソフトウェアは、ブラウザ内の印刷およびコピー機能を無効にし、学生がスクリーンショットを撮ったり、試験問題を印刷したりできないようにします。

**他のアプリケーションへのアクセスをブロック:**ブラウザーは、試験中に生徒が他のアプリケーションや Web サイトにアクセスするのを防ぎます。つまり、生徒はメッセージ アプリを使用したり、Web で回答を検索したりできません。

カスタマイズ可能な設定: Respondus LockDown ブラウザを使用すると、講師は特定のテスト ニーズに合わせて設定をカスタマイズできます。インストラクターは、ウェブカメラの使用を要求するか、数学の試験で電卓を有効にするかを選択できます。

**自動システム チェック:**ソフトウェアは各試験の前に自動システム チェックを実行し、学生が互換性のあるシステムを使用していること、およびブラウザが正しくインストールされていることを確認します。

試験セッションの記録とレビュー: Respondus LockDown ブラウザを使用すると、講師は試験セッションを記録およびレビューできるため、不正行為を防止するための追加レベルのセキュリティが提供されます。

互換性: Respondus LockDown ブラウザは、Windows と Mac の両方のオペレーティング システムと互換性があり、ほとんどの学習管理システムと統合されています。

よくある質問

Respondus LockDown ブラウザとは何ですか?

Respondus LockDown Browser は、学習管理システム (LMS) 内のテスト環境をロックダウンするカスタム ブラウザーです。他のアプリケーションへのアクセスをブロックし、コピー/貼り付けと印刷を無効にし、オンライン試験中にオーディオとビデオをキャプチャすることにより、不正行為を防止するように設計されています.

Respondus LockDown ブラウザをダウンロードするにはどうすればよいですか?

Respondus LockDown Browser は通常、教育機関の学習管理システムまたはオンライン ポータルからダウンロードできます。教育機関が許可している場合は、Respondus Web サイトから、または FileNUX 公式 Web サイトから直接ダウンロードすることもできます。

Respondus LockDown Browser はどのようにチートを防止しますか?

Respondus LockDown Browser は、他のアプリケーションへのアクセスをブロックし、コピー/貼り付けと印刷を無効にし、オンライン試験中にオーディオとビデオをキャプチャすることで、不正行為を防ぎます。また、学生が試験ページから移動したり、他の Web サイトにアクセスしたりすることも制限されます。

学生は、モバイル デバイスで Respondus LockDown Browser を使用できますか?

はい、Respondus LockDown Browser は iOS および Android デバイスで利用できます。ただし、オンライン試験でのモバイル デバイスの使用を許可するかどうかは、教育機関が決定します。

Respondus LockDown ブラウザでの試験中に学生のインターネット接続が失われた場合はどうなりますか?

Respondus LockDown Browser での試験中に学生のインターネット接続が失われた場合、学生はオフラインで試験に取り組み続けることができます。インターネット接続が復旧すると、生徒の回答が自動的に送信されます。

Respondus LockDown ブラウザをバイパスできますか?

ブラウザがインストールされていない別のコンピュータまたはデバイスを使用して、Respondus LockDown ブラウザをバイパスすることができます。ただし、これは学問的誠実性の侵害であり、深刻な結果を招く可能性があります。

Respondus LockDown Browser を試験以外の目的で使用できますか?

いいえ、Respondus LockDown ブラウザは、オンライン試験中に使用するために特別に設計されています。一般的な閲覧や他のウェブサイトへのアクセスには使用できません。

Respondus LockDown ブラウザのシステム要件は何ですか?

Respondus LockDown Browser のシステム要件は、使用しているオペレーティング システムと LMS によって異なります。通常、ブラウザは Windows および Mac オペレーティング システムと互換性があります。

講師は Respondus LockDown Browser を使用してどのように試験を設定できますか?

講師は、教育機関の学習管理システムから Respondus LockDown Browser を使用して試験を設定できます。通常、試験に Respondus LockDown Browser を要求するオプションを選択する必要があります。

Respondus LockDown Browser を対面試験に使用できますか?

いいえ、Respondus LockDown ブラウザはオンライン試験専用に設計されています。対面試験での使用は意図されていません。
