Microsoft Edge は、2015 年に

ダウンロード
安全、無料、高速

Microsoft Edge は、2015 年に最初にリリースされた Microsoft によって開発された Web ブラウザーです。これは、以前のバージョンの Windows の既定の Web ブラウザーであった Internet Explorer に代わる、最新で高速な代替になるように設計されています。

Microsoft Edge の顕著な特徴は、その速度です。最新の Web 標準に合わせて最適化された EdgeHTML と呼ばれる新しいレンダリング エンジンを使用しているため、他のブラウザーよりも高速かつ効率的にページを読み込むことができます。また、Edge にはクリーンでシンプルなインターフェイスがあり、気を散らすものを取り除き、読んでいるコンテンツに集中しやすくする読書モードなどの機能を備えています。

Microsoft Edge は、そのパフォーマンスとデザインに加えて、Web の閲覧中にユーザーを保護するためのさまざまなセキュリティおよびプライバシー機能も提供します。これらには、ポップアップや不要な Cookie をブロックするオプション、および強力なパスワードを生成して後で使用するために保存できる組み込みのパスワード マネージャーが含まれます。

Microsoft Edge のもう 1 つの注目すべき機能は、Microsoft の他の製品やサービスとの統合です。たとえば、ブラウザーを使用して OneDrive アカウントにアクセスしたり、注釈ツールを使用して Web ページをマークアップして他のユーザーと共有したりできます。Microsoft Edge は、ブラウザーに追加機能を追加する小さなプログラムである “拡張機能” もサポートしています。

まとめると、Microsoft Edge は高速で安全、かつ機能豊富な Web ブラウザーであり、個人用と業務用の両方に適しています。レジャーで Web を閲覧している場合でも、重要なドキュメントで作業している場合でも、Edge には仕事を完了するために必要なツールと機能が備わっています。

機能とハイライト

ユーザー インターフェイス: Microsoft Edge には、クリーンで最新の操作しやすいユーザー フレンドリーなインターフェイスが付属しています。シンプルなデザインで、ユーザーが見ているコンテンツに集中しやすくなっています。

タブ管理: Edge は優れたタブ管理機能を提供します。キーボード ショートカット、マウス クリック、またはタッチ ジェスチャを使用して、タブを簡単に開いたり、閉じたり、切り替えたりできます。また、現在使用していないが後でアクセスしたいタブを脇に置いておくこともできます。

閲覧表示: Microsoft Edge は、記事の本文と画像のみを残して、Web ページから広告や気を散らすものを削除する「閲覧表示」オプションを提供します。この機能により、Web ページが読みやすくなり、気が散らなくなります。

Cortana の統合: Cortana は Microsoft の仮想アシスタントであり、Edge は Cortana と統合してユーザーに追加機能を提供します。たとえば、Cortana に Web サイトの検索やトピックに関する情報の取得を手伝ってもらうことができます。

拡張機能: Edge は拡張機能をサポートしているため、ユーザーはブラウザーに機能を追加できます。広告ブロック、パスワード管理、ソーシャル メディアなどの拡張機能があります。

PDF サポート: Edge は、追加のソフトウェアを必要とせずに PDF ファイルを表示して注釈を付けることができます。この機能は、PDF ドキュメントを頻繁に使用する必要があるユーザーにとって特に便利です。

セキュリティ: Microsoft Edge には、悪意のある Web サイトや攻撃からユーザーを保護するのに役立ついくつかのセキュリティ機能があります。組み込みのフィッシング フィルター、マルウェア保護、およびオペレーティング システムからブラウザーを分離するサンドボックス テクノロジがあります。

詳しくは

Microsoft Edge は、Google Chrome の基礎でもあるオープンソース ブラウザー プロジェクトであるChromiumに基づいています。2018 年 12 月、Microsoft は、互換性とパフォーマンスを向上させるために、Chromium エンジンを使用して Edge を再構築すると発表しました。Edge の新しいバージョンは 2020 年 1 月にリリースされ、現在は Google Chrome と同じコア テクノロジに基づいています。

Microsoft Edge とGoogle Chrome は、Chromium で共通の基盤を共有していますが、独自の機能とデザインを備えた別個のブラウザーです。たとえば、Edge には Chrome にはない読み取りモードと注釈ツールが含まれており、Microsoft の他の製品やサービスとも統合されています。ただし、2 つのブラウザーは同じ拡張機能と Web アプリケーションを実行でき、同様のパフォーマンスとセキュリティ機能も備えています。

ブラウザーの戦い: トップ インターネット ナビゲーターの機知に富んだレビュー

インターネットの広大で無限の領域を探索するために信頼できるツールです。猫の動画を検索する場合でも、最新のハンドスピナーを購入する場合でも、Web ブラウザーは不可欠です。しかし正直に言うと、それらはすべて同じように作られているわけではありません。滑らかで速いものもあれば、日曜の朝のカタツムリのように不格好で遅いものもあります。

まず、万能の Google Chrome があります。それは丘の王様、トップ ドッグ、ブラウザーのビヨンセです。Chrome は非常に高速で、インターネット接続に必要なすべての機能を備えています。さらに、インターネットが切れたときに恐竜ゲームがポップアップする唯一のブラウザです。あなたのブラウザはそれを行うことができますか？そうは思いませんでした。

次に、 Mozilla Firefox があります。それは、常に Chrome に追いつこうとしている弱肉強食のようなものです。確かに、それほど速くはありませんが、心があります。Firefox はオープンソースです。つまり、Firefox は、誰もが無料でアクセスできるブラウザーを作成することに情熱を傾けている開発者のコ​​ミュニティによって作成されています。本当のことを言えば、小さなキツネのロゴは可愛すぎて抵抗できません。

次に、Microsoft Edge があります。これは、誰も求めていなかったにもかかわらず、私たちが手に入れたブラウザーです。以前は Internet Explorer と呼ばれていましたが、Microsoft は、月曜日の朝にナマケモノよりも遅いブラウザを使いたがる人はいないことに気付きました。そのため、彼らはそれをブランド変更し、フェイスリフトを行いました。悪くはないですが、素晴らしいものでもありません。それはブラウザーの中間子のようなもので、常にすべての人を喜ばせようとしますが、不十分です。

最後になりましたが、Safari があります。これは、クパチーノの甘い、甘い良さを十分に得ることができない Apple 狂信者のためのブラウザです。Mac や iPhone でインターネットを閲覧するために必要なすべての機能を備えており、高速でスマートです。しかし、本当のことを言いましょう。人々が Safari を使用する唯一の理由は、Chrome をダウンロードするのが面倒だからです。

これで、最も人気のある Web ブラウザを簡単にまとめました。あなたが Chrome の熱狂的なファンであろうと、Firefox の支持者であろうと、1 つのことに同意できます。結局のところ、一日中仕事を先延ばしにする方法が他にあるでしょうか?
