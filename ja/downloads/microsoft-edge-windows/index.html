Microsoft Edge は、2015 年に

ダウンロード
安全、無料、高速

Microsoft Edge は、2015 年に最初にリリースされた Microsoft によって開発された Web ブラウザーです。これは、以前のバージョンの Windows の既定の Web ブラウザーであった Internet Explorer に代わる、最新で高速な代替になるように設計されています。

Microsoft Edge の顕著な特徴は、その速度です。最新の Web 標準に合わせて最適化された EdgeHTML と呼ばれる新しいレンダリング エンジンを使用しているため、他のブラウザーよりも高速かつ効率的にページを読み込むことができます。また、Edge にはクリーンでシンプルなインターフェイスがあり、気を散らすものを取り除き、読んでいるコンテンツに集中しやすくする読書モードなどの機能を備えています。

Microsoft Edge は、そのパフォーマンスとデザインに加えて、Web の閲覧中にユーザーを保護するためのさまざまなセキュリティおよびプライバシー機能も提供します。これらには、ポップアップや不要な Cookie をブロックするオプション、および強力なパスワードを生成して後で使用するために保存できる組み込みのパスワード マネージャーが含まれます。

Microsoft Edge のもう 1 つの注目すべき機能は、Microsoft の他の製品やサービスとの統合です。たとえば、ブラウザーを使用して OneDrive アカウントにアクセスしたり、注釈ツールを使用して Web ページをマークアップして他のユーザーと共有したりできます。Microsoft Edge は、ブラウザーに追加機能を追加する小さなプログラムである “拡張機能” もサポートしています。

まとめると、Microsoft Edge は高速で安全、かつ機能豊富な Web ブラウザーであり、個人用と業務用の両方に適しています。レジャーで Web を閲覧している場合でも、重要なドキュメントで作業している場合でも、Edge には仕事を完了するために必要なツールと機能が備わっています。

機能とハイライト

ユーザー インターフェイス: Microsoft Edge には、クリーンで最新の操作しやすいユーザー フレンドリーなインターフェイスが付属しています。シンプルなデザインで、ユーザーが見ているコンテンツに集中しやすくなっています。

タブ管理: Edge は優れたタブ管理機能を提供します。キーボード ショートカット、マウス クリック、またはタッチ ジェスチャを使用して、タブを簡単に開いたり、閉じたり、切り替えたりできます。また、現在使用していないが後でアクセスしたいタブを脇に置いておくこともできます。

閲覧表示: Microsoft Edge は、記事の本文と画像のみを残して、Web ページから広告や気を散らすものを削除する「閲覧表示」オプションを提供します。この機能により、Web ページが読みやすくなり、気が散らなくなります。

Cortana の統合: Cortana は Microsoft の仮想アシスタントであり、Edge は Cortana と統合してユーザーに追加機能を提供します。たとえば、Cortana に Web サイトの検索やトピックに関する情報の取得を手伝ってもらうことができます。

拡張機能: Edge は拡張機能をサポートしているため、ユーザーはブラウザーに機能を追加できます。広告ブロック、パスワード管理、ソーシャル メディアなどの拡張機能があります。

PDF サポート: Edge は、追加のソフトウェアを必要とせずに PDF ファイルを表示して注釈を付けることができます。この機能は、PDF ドキュメントを頻繁に使用する必要があるユーザーにとって特に便利です。

セキュリティ: Microsoft Edge には、悪意のある Web サイトや攻撃からユーザーを保護するのに役立ついくつかのセキュリティ機能があります。組み込みのフィッシング フィルター、マルウェア保護、およびオペレーティング システムからブラウザーを分離するサンドボックス テクノロジがあります。

ユーザーインターフェース

Microsoft Edge のユーザー インターフェイスは、シンプルさと機能性に重点を置いて、クリーンでモダン、かつユーザー フレンドリーになるように設計されています。UI の主要な要素は次のとおりです。

アドレス バー:ブラウザー ウィンドウの上部にあるアドレス バーを使用すると、ユーザーは URL を入力したり、クエリを検索したりできます。Edge はデフォルトの検索エンジンとして Bing を使用しますが、ユーザーは設定でこれを好みの検索エンジンに変更できます。

タブ:アドレス バーの上で、ユーザーはタブ付きのインターフェイスを使用して複数の開いている Web ページを管理できます。各タブにはページのタイトルが表示され、利用可能な場合は簡単に識別できるファビコンが表示されます。

新しいタブ ページ:新しいタブを開くと、検索バー、頻繁にアクセスする Web サイト、ニュースや天気予報などのパーソナライズされたコンテンツを表示するカスタマイズ可能なスタート ページが表示されます。

ナビゲーション ボタン:アドレス バーの左側にある、標準の [戻る]、[進む]、および [更新] ボタンを見つけて、簡単に移動できるようにします。

メニュー:ウィンドウの右上隅にある 3 つのドットのアイコン (省略記号) をクリックするとメイン メニューが開き、ブックマーク、履歴、拡張機能、プライバシーとセキュリティのオプションなど、さまざまなブラウザー機能と設定にアクセスできます。 .

コレクション: Microsoft Edge の独自の機能であるコレクションを使用すると、ユーザーはリンク、画像、テキストなどの Web コンテンツを専用のサイドバーに集めて整理できます。この機能にアクセスするには、アドレス バーの右側にある [コレクション] アイコンをクリックします。

お気に入りとリーディング リスト:ユーザーはお気に入りの Web サイトを保存して後で簡単にアクセスできます。また、リーディング リスト機能を使用すると、記事や Web ページを保存して後で読むか、オフラインで読むことができます。

拡張機能: Microsoft Edge は、ブラウザーの機能を強化し、ユーザー エクスペリエンスをカスタマイズできる Chrome Web ストアの拡張機能を含む、幅広い拡張機能をサポートしています。

アクセシビリティ機能: Edge は、Web ページのレイアウトを簡素化し、テキスト読み上げ機能を提供するイマーシブ リーダー モードや、視覚障害のあるユーザーを支援するハイ コントラスト テーマなど、さまざまなアクセシビリティ機能を提供します。

Microsoft Edge のユーザー インターフェイスは、あらゆる経験レベルのユーザーが簡単にナビゲートできる、シームレスで効率的なブラウジング エクスペリエンスを提供するように設計されています。

タブ管理

Microsoft が開発した Web ブラウザーである Microsoft Edge には、ユーザー エクスペリエンスの向上を目的としたいくつかの機能が導入されており、タブ管理はその 1 つです。Microsoft Edge のタブ管理は、ユーザーが開いている複数のタブを効果的に整理、移動、処理できるように設計されています。

Microsoft Edge のタブ管理機能の重要な側面には、次のようなものがあります。

垂直タブ:この機能により、ユーザーは開いているタブを従来の水平レイアウトからウィンドウの横にある垂直スタックに移動できます。これにより、開いているタブの概要がわかりやすくなり、それらを簡単に見つけて切り替えることができます。垂直タブを有効にするには、ブラウザ ウィンドウの左上隅にある [垂直タブ] ボタンをクリックするだけです。

タブ グループ:タブ グループを使用すると、ユーザーは類似したタブを 1 つのグループに分類してラベルを付けることができます。これにより、ブラウザーが整理され、関連するタブ間を移動しやすくなります。タブ グループを作成するには、ユーザーはタブを右クリックし、[新しいグループにタブを追加] または [既存のグループにタブを追加] を選択します。

タブのプレビュー:この機能は、マウスをタブの上に置くと、タブの内容の小さなプレビューを提供します。これにより、ユーザーはタブに切り替える必要なく、タブのコンテンツをすばやく識別することができます。

スリープ タブ: Microsoft Edge は、システム リソースを節約するために、「スリープ タブ」と呼ばれる機能を導入しました。タブが一定時間使用されないと、自動的にスリープ モードになり、システム リソースが解放されます。ユーザーは、タブがスリープ状態になるまでの時間をカスタマイズできます。スリープ状態のタブは、クリックすると自動的にリロードされます。

タブのピン留め:ユーザーは、頻繁にアクセスする Web サイトや重要なタブをピン留めして、タブ行の先頭に永続的に表示することができます。これにより、最も重要なタブや頻繁に使用するタブに簡単にアクセスできます。

キーボード ショートカット: Microsoft Edge は、新しいタブを開くための Ctrl+T、現在のタブを閉じるための Ctrl+W、開いているタブを切り替えるための Ctrl+Tab など、タブ管理のためのさまざまなキーボード ショートカットをサポートしています。

Microsoft Edge のこれらのタブ管理機能により、全体的なブラウジング エクスペリエンスが大幅に向上し、複数のタブを一度に管理するユーザーにとってより効率的で整理されたものになります。

リーディング ビュー

Microsoft Edge のリーディング ビュー機能は、ユーザーのオンライン リーディング エクスペリエンスを向上させるために設計された便利なツールです。Web ページのレイアウトを簡素化し、広告、ポップアップ、その他の不要な要素などの気を散らすものを取り除き、Web ページのメイン コンテンツに集中しやすくします。

Microsoft Edge の閲覧表示機能の重要な側面を次に示します。

よりクリーンなレイアウト:広告、サイドバー、およびその他の混乱を取り除くことで、リーディング ビューは、ユーザーが読んでいるテキストに集中するのに役立つ、クリーンで整理されたインターフェイスを提供します。これは、長い記事や内容の濃いコンテンツを読むときに特に役立ちます。

カスタマイズ:ユーザーは、さまざまなフォント スタイル、テキスト サイズ、および背景色から選択して、閲覧ビューの外観をカスタマイズできます。これにより、個人の好みやアクセシビリティのニーズに応える、パーソナライズされた読書体験が可能になります。

テキスト読み上げ:この機能は、コンテンツを読み上げる「読み上げ」と呼ばれる組み込みのテキスト読み上げ機能を提供します。これは、視覚障害のあるユーザーや、コンテンツを読むよりも聞くことを好むユーザーにとって役立ちます。

オフラインでの閲覧:閲覧ビューを使用すると、ユーザーはオフラインで閲覧するために Web ページを保存できるため、インターネット接続が利用できない場合でもコンテンツに簡単にアクセスできます。

簡単なアクセス:アドレス バーにある本のアイコンを使用するか、キーボード ショートカットの Ctrl+Shift+R を押すと、閲覧表示機能のオンとオフを簡単に切り替えることができます。これにより、必要に応じて標準ビューと簡易ビューを簡単に切り替えることができます。

Microsoft Edge の閲覧ビューは、オンライン コンテンツを読むときのユーザー エクスペリエンスを向上させるための貴重なツールです。カスタマイズ オプションとアクセシビリティ機能を提供することで、幅広いユーザーのニーズと好みに対応します。

コルタナ統合

Microsoft Edge は、同社の仮想アシスタントである Cortana を統合して、ユーザー エクスペリエンスを強化し、よりシームレスなブラウジング エクスペリエンスを提供します。Cortana 統合機能として知られるこの統合により、ユーザーは Edge ブラウザー内で仮想アシスタントを直接操作できます。Cortana は最初に Windows 10 で導入されましたが、その後、他のプラットフォームやデバイスでも利用できるようになりました。

Microsoft Edge での Cortana の統合には、次のようないくつかの利点と機能があります。

音声検索:ユーザーはアドレス バーから直接 Cortana を使用して音声検索を実行できます。この機能により、ユーザーはクエリを手動で入力しなくても情報を検索できます。

コンテキスト情報: Cortana は、Web ページで選択されたテキストのコンテキスト情報を提供できます。テキストを強調表示して右クリックすることにより、ユーザーは Cortana の支援にアクセスし、現在のページを離れずに関連情報や定義を取得できます。

パーソナライズされた提案: Cortana は、ユーザーの閲覧履歴、好み、興味に基づいてパーソナライズされた提案を提供できます。これにより、ユーザーは新しいコンテンツを見つけやすくなり、特定の情報を検索するときに時間を節約できます。

リマインダーと通知: Cortana は、Microsoft Edge ブラウザーから直接リマインダーと通知を送信できます。ユーザーは、別のアプリやウィンドウを開くことなく、リマインダーを設定したり、重要なアラートを受信したりできます。

他の Microsoft サービスとの統合: Cortana は、Outlook、OneNote、To-Do などの他の Microsoft サービスとシームレスに連携するため、ユーザーはタスク、予定、メモをブラウザーから直接管理できます。

クロスデバイス同期: Cortana はデバイス間で情報を同期できるため、ユーザーはスマートフォン、タブレット、または PC からリマインダー、メモ、およびその他のデータにアクセスできます。

Microsoft はその後、スタンドアロンの仮想アシスタントとしての Cortana から、Microsoft エコシステム内のより具体的な生産性シナリオに統合することに焦点を移したことに注意することが重要です。したがって、仮想アシスタントの可用性と機能は、地域とデバイスによって異なる場合があります。

ただし、Microsoft Edge の Cortana 統合機能は、ユーザーにさまざまな利点と利便性を提供し、全体的なブラウジング エクスペリエンスを向上させます。

拡張機能のサポート

拡張機能は、ブラウザーの機能を変更および強化できる小さなソフトウェア プログラムであり、ユーザーがニーズや好みに合わせてブラウジング エクスペリエンスを調整できるようにします。

Microsoft Edge の拡張機能のサポートには、いくつかの利点があります。

幅広い拡張機能: Microsoft Edge は Chromium に基づいているため、Chrome Web ストアで入手できる Chrome 拡張機能の大部分をサポートできます。これにより、選択可能なオプションの大規模で確立されたライブラリが既に存在するため、ユーザーはニーズに合った拡張機能を簡単に見つけてインストールできます。

簡単なインストール: Microsoft Edge に拡張機能をインストールするのは非常に簡単です。ユーザーは、Microsoft Edge アドオン Web サイトまたは Chrome Web ストアにアクセスして、拡張機能を参照してインストールできます。目的の拡張機能が見つかったら、[Chrome に追加] または [取得] ボタンをクリックしてインストールすると、拡張機能がブラウザに追加されます。

拡張機能の管理: Microsoft Edge は、インストールされた拡張機能を管理するための使いやすいインターフェイスを提供します。ユーザーは、ブラウザーの右上隅にある 3 つのドットのメニュー アイコンをクリックし、[拡張機能] を選択して、拡張機能ページにアクセスできます。そこから、ユーザーは拡張機能を有効化、無効化、または削除したり、個々の拡張機能の設定にアクセスしたりできます。

セキュリティとプライバシー: Microsoft は、ユーザーのセキュリティとプライバシーを確​​保することを約束しており、これは拡張機能のサポートにも及びます。Microsoft Edge には、アドオン Web サイトに送信された拡張機能のレビュー プロセスがあり、ユーザーが利用できるようになる前に特定の基準を満たしていることを確認します。さらに、ユーザーは各拡張機能に付与されるアクセス許可を制御して、アクセスできるデータを制限できます。

開発者サポート: Microsoft は、Microsoft Edge の拡張機能の作成に関心のある開発者向けのリソースとドキュメントを提供しています。開発者は、Microsoft Edge Extension Toolkit を使用して、既存の Chrome 拡張機能を変換したり、新しい拡張機能を最初から作成したりできます。このサポートにより、拡張エコシステムの成長が促進され、ユーザーがさまざまなツールや機能にアクセスできるようになります。

PDF サポート

Microsoft Edge の PDF 機能により、ユーザーは、サードパーティ ソフトウェアを必要とせずに、PDF ファイルを表示、ナビゲート、注釈付け、および管理するための便利な方法を利用できます。PDF ドキュメントを操作するために Microsoft Edge が提供する主な機能の一部を次に示します。

PDF の表示: Microsoft Edge を使用すると、ユーザーはブラウザー内で直接 PDF ファイルを開いて表示できます。ファイルには、ローカル ドライブまたは URL からアクセスできます。

ナビゲーション:ブラウザーは、ドキュメントのサムネイルを表示するためのサイドバー、目次、ブックマーク、ページ番号を入力して特定のページにジャンプする機能など、直感的なナビゲーション ツールを提供します。

ズームと表示オプション: Edge は、読みやすさを向上させ、ユーザーの好みに対応するために、さまざまなズーム レベルと表示オプション (1 ページ、連続スクロール、2 ページ表示など) を提供します。

注釈:ユーザーは PDF テキストにコメント、ハイライト、下線、および取り消し線を追加できるため、文書を読みながら共同作業やメモを取ることが容易になります。さらに、ユーザーは図形やテキスト ボックスを PDF に描画または追加できます。

テキストの選択とコピー: Microsoft Edge を使用すると、ユーザーは PDF ファイルからテキストを選択してコピーし、他のアプリケーションに貼り付けることができます。

検索:ブラウザーには、ユーザーが PDF ドキュメント内の特定のテキストまたはフレーズを検索できる組み込みの検索機能が含まれています。

フォームへの入力と署名: Edge は PDF フォームへの入力と署名をサポートしているため、ユーザーはブラウザー内で直接さまざまな種類のフォームに入力して送信できます。

アクセシビリティ: Microsoft Edge は、スクリーン リーダーなどの支援技術の組み込みサポートを提供し、視覚障害のあるユーザーのアクセシビリティを向上させるために、ハイ コントラスト モードを有効にしたり読み上げたりするオプションを提供します。

保存と印刷:ユーザーは、PDF ファイルをローカル ドライブに簡単に保存したり、ブラウザから直接印刷したりできます。

他の Microsoft サービスとの統合: Edge は、OneDrive、SharePoint、Outlook などの他の Microsoft サービスとシームレスに統合されるため、さまざまなプラットフォームで PDF ファイルを簡単に共有、アクセス、管理できます。

Microsoft Edge は PDF ドキュメントを操作するための堅牢な機能セットを提供しますが、専用の PDF ソフトウェアまたはサードパーティのブラウザー拡張機能が提供するすべての高度な機能が含まれていない場合があります。しかし、ほとんどのユーザーにとって、Edge の組み込みの PDF サポートは、日常のタスクやドキュメント管理には十分すぎるはずです.

セキュリティ機能

Microsoft Edge は、ユーザーの安全なブラウジング エクスペリエンスを確保するために、セキュリティに重点を置いています。Microsoft Edge に含まれる主要なセキュリティ機能の一部は次のとおりです。

SmartScreen フィルター: SmartScreen は Microsoft が開発した機能で、ユーザーをフィッシング攻撃、マルウェア、およびその他の悪意のあるソフトウェアから保護するのに役立ちます。Web サイトとファイルの URL を、報告された悪意のあるサイトとダウンロードの常に更新されるリストと照合してチェックし、ユーザーが潜在的に危険な Web サイトにアクセスしようとしているか、または潜在的に有害なファイルをダウンロードしようとしている場合に警告します。

サンドボックス化: Edge は、Web コンテンツをサンドボックス化されたプロセスに分離するマルチプロセス アーキテクチャを使用します。つまり、悪意のある Web サイトがブラウザーの脆弱性を悪用した場合、被害は限定的であり、攻撃者はユーザーの個人データやその他のシステム リソースにアクセスできなくなります。

Microsoft Defender の統合: Microsoft Edge には、同社のウイルス対策およびマルウェア対策ソリューションである Microsoft Defender との統合が組み込まれています。この統合により、ブラウザからアクセスされたダウンロードとファイルのリアルタイム スキャンが提供され、有害な可能性のあるファイルを検出してブロックするのに役立ちます。

追跡防止: Microsoft Edge は追跡防止オプションを提供し、ユーザーがオンライン プライバシーをより詳細に制御できるようにします。ユーザーは、基本、バランス、厳格の 3 つの異なるトラッキング防止レベルから選択できます。各レベルは異なるカテゴリのトラッカーをブロックし、厳密な設定では最高レベルのプライバシー保護が提供されます。

InPrivate ブラウズ:他のブラウザーのプライベート ブラウジング モードと同様に、Edge の InPrivate ブラウズを使用すると、ユーザーは、閲覧履歴、インターネット一時ファイル、または Cookie をデバイスに保存することなく、Web を閲覧できます。

安全な更新: Microsoft Edge は、セキュリティ パッチと改善により定期的に更新されます。これらの更新プログラムは Windows Update を通じて安全に配信され、ユーザーが最新の脅威から確実に保護されるようにします。

パスワード モニター:この機能は、ユーザーが保存したパスワードが既知のデータ侵害に関与した場合に警告することで、オンライン アカウントを安全に保つのに役立ちます。次に、ユーザーは、アカウントのセキュリティを維持するために、影響を受けるパスワードを変更するように求められます。

既定で HTTPS: Microsoft Edge は、利用可能な場合、Web サイトへの接続を HTTP から HTTPS に自動的にアップグレードし、より安全で暗号化された接続を提供して、ユーザーのデータを保護します。

よくある質問

Microsoft Edge とは何ですか?

Microsoft Edge は、Microsoft が開発した Web ブラウザーで、Windows、macOS、Android、および iOS で利用できます。

Microsoft Edge が最初にリリースされたのはいつですか?

Microsoft Edge は、Windows 10 の一部として、2015 年 7 月 29 日に最初にリリースされました。

Microsoft Edge の前身は何ですか?

Microsoft Edge の前身は Internet Explorer でした。

Microsoft Edge の機能にはどのようなものがありますか?

Microsoft Edge の機能には、カスタマイズ可能なスタート ページ、拡張機能、閲覧モード、組み込みの PDF リーダー、Cortana (Microsoft の仮想アシスタント) との統合などがあります。

Microsoft Edge はモバイル デバイスで使用できますか?

はい。Microsoft Edge は、Android や iOS などのモバイル デバイスで使用できます。

Microsoft Edge の既定の検索エンジンは何ですか?

Microsoft Edge の既定の検索エンジンは Bing ですが、ユーザーは必要に応じて別の検索エンジンに変更できます。

Microsoft Edge のリーディング リスト機能とは何ですか?

Microsoft Edge のリーディング リスト機能を使用すると、ユーザーは Web ページを保存して後で読むことができます。

Microsoft Edge の “ハブ” とは何ですか?

Microsoft Edge のハブは、ブックマーク、リーディング リスト、履歴、およびダウンロードへのアクセスを提供するサイドバーです。

Microsoft Edge は拡張機能をサポートしていますか?

はい。Microsoft Edge は、 Microsoft Storeからダウンロードできる拡張機能をサポートしています。

Microsoft Edge はデバイス間でデータを同期できますか?

はい、Microsoft Edge は、ブックマーク、履歴、パスワード、フォーム データなど、デバイス間でデータを同期できます。

Microsoft Edge のコレクション機能とは何ですか?

Microsoft Edge のコレクション機能を使用すると、ユーザーは Web コンテンツをグループに保存して整理できます。

Microsoft Edge は Web 標準と互換性がありますか?

はい。Microsoft Edge は、HTML5、CSS3、JavaScript などの Web 標準と互換性があります。

Microsoft Edge DevTools とは何ですか?

Microsoft Edge DevTools は、開発者が Web ページのデバッグとテストに使用できる一連のツールです。

Microsoft Edge は広告をブロックできますか?

はい、Microsoft Edge は、組み込みの追跡防止機能を使用して広告をブロックできます。

Microsoft Edge は無料で使用できますか?

はい、Microsoft Edge は無料で使用でき、Windows デバイスにプレインストールされています。

詳しくは

Microsoft Edge は、Google Chrome の基礎でもあるオープンソース ブラウザー プロジェクトであるChromiumに基づいています。2018 年 12 月、Microsoft は、互換性とパフォーマンスを向上させるために、Chromium エンジンを使用して Edge を再構築すると発表しました。Edge の新しいバージョンは 2020 年 1 月にリリースされ、現在は Google Chrome と同じコア テクノロジに基づいています。

Microsoft Edge とGoogle Chrome は、Chromium で共通の基盤を共有していますが、独自の機能とデザインを備えた別個のブラウザーです。たとえば、Edge には Chrome にはない読み取りモードと注釈ツールが含まれており、Microsoft の他の製品やサービスとも統合されています。ただし、2 つのブラウザーは同じ拡張機能と Web アプリケーションを実行でき、同様のパフォーマンスとセキュリティ機能も備えています。

ブラウザーの戦い: トップ インターネット ナビゲーターの機知に富んだレビュー

インターネットの広大で無限の領域を探索するために信頼できるツールです。猫の動画を検索する場合でも、最新のハンドスピナーを購入する場合でも、Web ブラウザーは不可欠です。しかし正直に言うと、それらはすべて同じように作られているわけではありません。滑らかで速いものもあれば、日曜の朝のカタツムリのように不格好で遅いものもあります。

まず、万能の Google Chrome があります。それは丘の王様、トップ ドッグ、ブラウザーのビヨンセです。Chrome は非常に高速で、インターネット接続に必要なすべての機能を備えています。さらに、インターネットが切れたときに恐竜ゲームがポップアップする唯一のブラウザです。あなたのブラウザはそれを行うことができますか？そうは思いませんでした。

次に、 Mozilla Firefox があります。それは、常に Chrome に追いつこうとしている弱肉強食のようなものです。確かに、それほど速くはありませんが、心があります。Firefox はオープンソースです。つまり、Firefox は、誰もが無料でアクセスできるブラウザーを作成することに情熱を傾けている開発者のコ​​ミュニティによって作成されています。本当のことを言えば、小さなキツネのロゴは可愛すぎて抵抗できません。

次に、Microsoft Edge があります。これは、誰も求めていなかったにもかかわらず、私たちが手に入れたブラウザーです。以前は Internet Explorer と呼ばれていましたが、Microsoft は、月曜日の朝にナマケモノよりも遅いブラウザを使いたがる人はいないことに気付きました。そのため、彼らはそれをブランド変更し、フェイスリフトを行いました。悪くはないですが、素晴らしいものでもありません。それはブラウザーの中間子のようなもので、常にすべての人を喜ばせようとしますが、不十分です。

最後になりましたが、Safari があります。これは、クパチーノの甘い、甘い良さを十分に得ることができない Apple 狂信者のためのブラウザです。Mac や iPhone でインターネットを閲覧するために必要なすべての機能を備えており、高速でスマートです。しかし、本当のことを言いましょう。人々が Safari を使用する唯一の理由は、Chrome をダウンロードするのが面倒だからです。

これで、最も人気のある Web ブラウザを簡単にまとめました。あなたが Chrome の熱狂的なファンであろうと、Firefox の支持者であろうと、1 つのことに同意できます。結局のところ、一日中仕事を先延ばしにする方法が他にあるでしょうか?
